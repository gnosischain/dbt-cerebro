api_mixpanel_ga_modals_daily

Not exposed via cerebro-api; accessible to the internal MCP / semantic layer. Daily open counts for each modal/bottom-sheet component in Gnosis App. Sourced from int_mixpanel_ga_modals_daily. Filter by bottom_sheet component name. Known bottom_sheet values: PasskeyStepsModal (passkey onboarding), ModalJoin (join Circles), SelectSwapAsset (swap token picker), ModalReceive (receive funds QR), ModalSend (send funds flow), CardActivationStepsModal (Gnosis Pay card activation). Use case: front-end engagement analysis — which entry points users interact with most.
