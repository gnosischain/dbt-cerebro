int_execution_transactions_by_project_alltime_state

This model aggregates execution transaction data by project and month, providing insights into transaction volume, fees, and user activity for comprehensive project-level analysis over time.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/transactions/intermediate/int_execution_transactions_by_project_alltime_state.sql

Tags: production execution transactions

Columns

NameTypeDescription
aa_stateStringA bitmap representing unique active users or accounts involved in transactions for the project during the month.
fee_stateFloat64The total sum of native fees associated with transactions for the project in the month, measured in the native currency units.
monthDateTime64The start date of the month for which the aggregated data is calculated, formatted as a date.
projectStringIdentifier for the project associated with the transaction data.
txs_stateUInt64The total number of transactions for the project within the month, represented as a summed state value.

Dimensions

  • month (time)
  • project (categorical)
  • aa_state (categorical)

Measures

  • txs_state_value (sum)
  • fee_state_value (sum)
