api_execution_live_trades_stats

One-row summary for the live-trades tab header tiles. Counts trades, sums USD volume, counts unique traders, and computes aggregator/multi-hop shares. Reads from `api_execution_live_trades` so the tiles exactly match the population users see in the table (same window, same dust filter, same multi-hop collapse).
