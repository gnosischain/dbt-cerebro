int_execution_tokens_balances_daily

The int_execution_tokens_balances_daily model tracks the daily balances of tokens for various addresses, incorporating net delta changes to maintain an up-to-date view of token holdings over time.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/tokens/intermediate/int_execution_tokens_balances_daily.sql

Tags: production execution tokens balances_daily

Columns

NameTypeDescription
addressStringThe blockchain address holding the token balance.
block_timestampDateTime64(0, 'UTC')UTC timestamp of the enclosing block.
chain_idUInt64EVM chain identifier.
dateDateThe specific date for which the token balances are recorded, representing a daily snapshot.
from_value_binaryStringBalance before change as 32-byte binary.
from_value_f64Float64Balance before change as floating-point ether.
from_value_stringStringBalance before change as decimal string.
insert_versionUInt64Monotonic ingestion version (nanoseconds).
net_delta_rawFloat64The raw net change in token quantity for the address on the given date, measured in token units.
symbolStringThe token's symbol, serving as a human-readable identifier.
to_value_binaryStringBalance after change as 32-byte binary.
to_value_f64Float64Balance after change as floating-point ether.
to_value_stringStringBalance after change as decimal string.
token_addressStringThe blockchain address of the token contract associated with the balance.
token_classStringThe classification or category of the token, such as ERC20 or other types.
