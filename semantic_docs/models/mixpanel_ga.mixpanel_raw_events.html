mixpanel_ga.mixpanel_raw_events

Raw Mixpanel events for the Gnosis App, ingested into ClickHouse. Each row is one event with a JSON properties blob containing browser, geo, referrer, page, and device metadata. The distinct_id column contains wallet addresses or device-scoped anonymous IDs and must never be exposed raw in downstream models.

Module: mixpanel_ga
