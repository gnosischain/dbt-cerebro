contracts_zodiac_modules_registry

ABI-resolution registry for Zodiac Modifier proxies (Delay + Roles) on Gnosis Chain, chain-wide (NOT GP-scoped). Mirrors contracts_gpay_modules_registry but selects every module proxy from int_execution_zodiac_module_proxies whose master_copy is one of the vulnerable Modifier mastercopies — the primary GP Delay (0x4a97e651...), the Roles mastercopy (0x9646fdad...), and the two older Delay implementations (0xd54895b1..., 0xd62129bf...) that share the same vulnerable codebase and still custody funds. Exposes abi_source_address (= master_copy) so decode_logs can resolve the mastercopy ABI for proxy-emitted events. Consumers that want only the primary GP Delay filter on master_copy / abi_source_address directly.
