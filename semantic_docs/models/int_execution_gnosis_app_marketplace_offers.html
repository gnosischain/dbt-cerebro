int_execution_gnosis_app_marketplace_offers

Gnosis App marketplace offers — one row per non-excluded PaymentGateway created via Circles v2 PaymentGatewayFactory. Sources & join: 1. createGateway calls from contracts_circles_v2_PaymentGatewayFactory_calls (carries the `name` string input as offer_name). 2. GatewayCreated events from contracts_circles_v2_PaymentGatewayFactory_events (carries the resulting `gateway` address and `owner`). 3. Joined on transaction_hash — createGateway emits GatewayCreated in the same tx — to pair (offer_name ↔ gateway_address). Exclusion: seeds/gnosis_app_marketplace_offers_excluded.csv. Rows match on offer_name OR gateway_address (empty string = "ignore this column for this row"). Dynamic-inclusion semantic: every new offer automatically appears in this model unless explicitly blocklisted — ops team curates the seed when a test gateway needs to be hidden.
