api_quarterly_data_circles_backers

Quarterly currently-trusted Circles backers, as of quarter end (revocation-aware). A backer is an address currently trusted by the backers group avatar var('circles_target_group_address'); trust is revocable, so this count can fall. Distinct from the ever-backed cumulative series (api:circles_v2_backers_cumulative).
