fct_execution_tokens_top_holders_latest

Latest-day snapshot of token holders ranked by USD balance, with concentration metrics (pct_of_total, cumulative_pct), 7-day balance change, and address labels from the crawlers pipeline. One row per (token_address, address). Top-N reflects ultimate beneficial owners — pool contracts are replaced with individual end-holders via the upstream UBO pipeline; all implemented phases (Aave, Balancer, Uniswap V3, Swapr V3) plug in with no change here. The 7-day delta is computed by driving the lookup FROM fct_execution_tokens_top_holders_ranked (small — top 500 per token) into 1-day slices of int_execution_tokens_balances_daily and fct_ubo_supply_claims_daily. ClickHouse builds a hash map from the ranked addresses and probes the 7d-ago partition, so only the matching rows are ever aggregated — avoiding the full-holder scan that previously caused OOM at 10.8 GiB.

Columns

NameTypeDescription
addressStringHolder wallet or contract address.
balanceFloat64Token balance in native units.
balance_usdFloat64Token balance in USD.
change_usd_7dFloat64USD balance change vs 7 days ago. Positive = accumulating, negative = exiting. For unwound rows the comparison is against the same UBO's lent balance 7 days ago.
cumulative_pctFloat64Running sum of pct_of_total from rank 1 downward — shows what share the top N holders control.
is_terminal_uboNullable(UInt8)1 = label-confirmed end-holder (EOA, Safe/AA, bridge, gpay). 0 = labeled but undecomposed container — Phase 2+ target. NULL = unclassified.
labelStringProject label from crawlers_data_labels (e.g. Safe, Balancer, EOA). NULL if unlabeled.
label_sectorStringSector from crawlers_data_labels (e.g. Wallets & AA, DEX, EOAs). NULL if unlabeled.
pct_of_totalFloat64Holder's balance as a percentage of total token supply (by USD), rounded to 4 decimals.
rankUInt64Holder rank within token, ordered by balance_usd descending.
symbolStringToken symbol (e.g. sDAI, WETH, GNO).
token_addressStringContract address of the token (lowercase).
token_classStringToken classification (STABLECOIN, OTHERS).
unwound_fromArray(String)Array of container addresses this row was unwound from (e.g. ['agnoSDAI']). Empty array means the holder is direct, not unwound from any container.
