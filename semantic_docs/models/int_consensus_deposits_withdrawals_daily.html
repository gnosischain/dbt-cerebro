int_consensus_deposits_withdrawals_daily

This model aggregates daily consensus deposit and withdrawal data, providing insights into the total amounts and transaction counts for each day, segmented by deposit and withdrawal types.

Module: consensus

Status: approved

Owner: analytics_team

Path: models/consensus/intermediate/int_consensus_deposits_withdrawals_daily.sql

Tags: production consensus deposits

Columns

NameTypeDescription
cntUInt64The total number of deposit or withdrawal transactions recorded on that day.
dateDateTime64The specific day for which the deposit or withdrawal data is aggregated, represented as a date.
labelStringEither 'Deposits' or 'Withdrawals' — identifies which side of the flow this row aggregates.
total_amountUInt64Total deposit or withdrawal amount for the day, in real GNO. FIXED 2026-07: the raw beacon-chain amount is Gwei-of-mGNO (Gnosis Beacon Chain mirrors Ethereum's 32-unit-per-validator convention; 32 mGNO = 1 real GNO — see the unit warning in int_consensus_validators_income_daily.sql for the full explanation). The SQL divides by 10^9 (Gwei -> mGNO) then by 32 (mGNO -> real GNO). Confirmed empirically: the modal raw deposit amount is exactly 32,000,000,000, and common values are clean multiples of it (real depositors making round 1/2/9/10/12-GNO deposits).
