fct_consensus_consolidations_daily

Daily EIP-7251 consolidation event counts and transferred amounts stacked by role ('self', 'source', 'target'). Materialized as a table so the dashboard API returns fast; source is small (~5k rows) so rebuild is instant. See int_consensus_validators_consolidations_daily for the application-day inference and amount derivation (source's last non-zero effective balance before the application day).
