int_esg_dynamic_power_consumption

The int_esg_dynamic_power_consumption model calculates estimated power consumption metrics for different node categories across geographic regions, incorporating empirical data, client efficiency, and PUE factors to support ESG and power usage analysis.

Module: ESG

Status: candidate

Owner: analytics_team

Path: models/ESG/intermediate/int_esg_dynamic_power_consumption.sql

Tags: production esg power_consumption
