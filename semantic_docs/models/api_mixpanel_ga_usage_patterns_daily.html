api_mixpanel_ga_usage_patterns_daily

Not exposed via cerebro-api; accessible to the internal MCP / semantic layer. Daily Gnosis App event counts by hour-of-day and ISO day-of-week (both UTC). One row per (date, hour_of_day, day_of_week) bucket. Sourced from int_mixpanel_ga_usage_patterns_daily. Use case: heat-map of when users are active during the week.
