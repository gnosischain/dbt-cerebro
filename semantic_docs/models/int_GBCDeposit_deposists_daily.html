int_GBCDeposit_deposists_daily

The `int_GBCDeposit_deposists_daily` view aggregates daily deposit amounts and withdrawal credentials from GBC deposit events to support financial analysis and reporting.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/GBCDeposit/intermediate/int_GBCDeposit_deposists_daily.sql

Tags: production execution GBCDeposit

Columns

NameTypeDescription
amountUInt64The total deposit amount in unsigned 64-bit integer units, summed for each date and withdrawal credential.
dateDateTime64The date (at day granularity) of the deposit event, representing the start of the day for aggregation.
withdrawal_credentialsStringThe withdrawal credentials associated with each deposit, indicating the destination address or account.
