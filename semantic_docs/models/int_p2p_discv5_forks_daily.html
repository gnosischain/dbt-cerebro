int_p2p_discv5_forks_daily

The `int_p2p_discv5_forks_daily` model tracks daily counts of peers associated with specific current and next forks in the Discv5 protocol, facilitating analysis of fork adoption and transition trends over time.

Module: p2p

Status: candidate

Owner: analytics_team

Path: models/p2p/intermediate/int_p2p_discv5_forks_daily.sql

Tags: production p2p discv5

Columns

NameTypeDescription
cntUInt64Count of peer visits for the given date, label, and fork.
dateDateTime64The specific day the peer data was recorded, represented as the start of the day.
forkStringThe name of the current fork the peer is using at the end of the day.
labelStringLabel indicating whether the record is for current or next fork.
next_forkStringThe name of the fork the peer is transitioning to or planning to adopt next.
peer_idUInt64A unique identifier for each peer participating in the network.

Dimensions

  • date (time)
  • peer_id (categorical)
  • fork (categorical)
  • next_fork (categorical)
  • label (categorical)

Measures

  • cnt_value (sum)
