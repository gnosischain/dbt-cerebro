api_esg_cif_network_vs_countries_daily

This view provides daily effective carbon intensity metrics for the network and selected countries, enabling comparison of carbon footprint performance over time.

Module: ESG

Status: candidate

Owner: analytics_team

Path: models/ESG/marts/api_esg_cif_network_vs_countries_daily.sql

Tags: production esg tier1 api:carbon_intensity_factors granularity:daily

Columns

NameTypeDescription
carbon_intensityFloat64The measured carbon intensity value in grams of CO2 equivalent per kWh, rounded to one decimal place.
dateDateThe specific date for the recorded carbon intensity data, formatted as YYYY-MM-DD.
entity_codeStringIdentifier for the data source or country, such as 'GNOSIS' for the network or country codes like 'USA' and 'DEU'.
