int_execution_circles_v2_invite_funnel

One row per invited human, carrying the mint-cadence signals for the five-stage funnel (Invited → ≥2 days → ≥7 days → ≥14 days → Active Minter). Skips the trivial acceptance mint by tracking days actually minted in the first 30 / 60 days post-invite and the date the avatar first reached canonical Active Minter status.
