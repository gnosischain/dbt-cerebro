int_execution_gnosis_app_weekly_earners

Weekly "economic earners" set for the WEAU metric, scoped to the Gnosis App: addresses that earned >= 1 gCRC cashback OR >= 1 CRC inviter fee in a given week, filtered to in-app users/actions. Consumes the circles-first ecosystem layer (int_execution_circles_v2_economically_active_avatars_weekly), keeping only Gnosis App users (int_execution_gnosis_app_users_current) and additionally requiring inviter fees to have come through a Gnosis App relayer tx (any_in_app_tx = 1). DEFINITION CHANGE vs the original global version: earnings via other apps / direct on-chain no longer count here — they remain visible in the circles-first layer. Full-rebuild table.
