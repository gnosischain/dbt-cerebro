fct_celo_gpay_balances_safe_daily

DENSE per-Safe/token/day net-flow balance (Celo analog of Gnosis Chain's int_execution_gpay_balances_user_daily). Densified over a date spine so downstream daily totals and latest-day snapshots are correct on every day. Net flow = (Top-up + Reversal) - (Payment + Withdrawal). Equals the true on-chain balance ONLY for USDC/USDT under the documented scope assumptions (see file header). Not a chain-wide balance snapshot.
