int_p2p_discv4_clients_daily

The `int_p2p_discv4_clients_daily` model aggregates peer discovery data for the discv4 protocol, providing daily metrics on peer attributes to monitor network participation and diversity.

Module: p2p

Status: candidate

Owner: analytics_team

Path: models/p2p/intermediate/int_p2p_discv4_clients_daily.sql

Tags: production p2p discv4

Columns

NameTypeDescription
clientStringThe client software used by the peer, with 'Unknown' if not specified.
dateDateTime64The specific day the data was recorded, representing the start of the day.
generic_providerStringThe provider or source of the peer, indicating its origin or affiliation.
labelStringLabel for the metric, e.g., client name, platform, provider, or country.
metricStringType of metric, e.g., 'Clients', 'Platform', 'Provider', 'Country'.
peer_countryStringThe country where the peer is located, based on IP geolocation.
peer_idUInt64A unique identifier for each peer in the network.
platformStringThe platform (e.g., OS or environment) on which the peer's client runs.
valueUInt64Count of peers corresponding to the metric and label.
