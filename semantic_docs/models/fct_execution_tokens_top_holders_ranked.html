fct_execution_tokens_top_holders_ranked

Latest-day snapshot of token holders ranked by USD balance with pct_of_total, filtered to top 500 per token. No joins — only window functions on int_execution_tokens_balances_daily. Materialized as a table to decouple the heavy scan from the lightweight joins in fct_execution_tokens_top_holders_latest.
