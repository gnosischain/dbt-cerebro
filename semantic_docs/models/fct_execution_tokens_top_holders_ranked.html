fct_execution_tokens_top_holders_ranked

Latest-day snapshot of top-500 token holders per token ranked by USD balance. Combines int_execution_tokens_balances_daily (direct holders, with known containers excluded via LEFT ANTI JOIN on fct_ubo_known_containers_daily) and fct_ubo_supply_claims_daily (UBO-unwound holders) for the single latest date. Uses scalar subqueries in WHERE for partition pruning rather than CROSS JOIN, keeping the scan to one date partition. Materialized as a table so fct_execution_tokens_top_holders_latest can drive its 7d-ago lookup from this small result set instead of re-scanning all holders.
