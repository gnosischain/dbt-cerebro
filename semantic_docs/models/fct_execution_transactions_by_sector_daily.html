fct_execution_transactions_by_sector_daily

The fct_execution_transactions_by_sector_daily model aggregates daily execution transaction data by sector to support business analysis of transaction volume, gas consumption, and fees across different sectors.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/transactions/marts/fct_execution_transactions_by_sector_daily.sql

Tags: production execution transactions

Columns

NameTypeDescription
active_accountsUInt64The bitmap-encoded set of unique active account identifiers for the given date and sector.
dateDateThe specific calendar date for the aggregated data, formatted as YYYY-MM-DD.
fee_native_sumFloat64The total native currency fees accumulated from transactions, rounded to six decimal places.
gas_used_sumFloat64The sum of gas used across all transactions for the given date and sector.
sectorStringThe sector or business area associated with the transactions, represented as a string.
txsUInt64The total number of transactions executed on the specified date and sector.

Dimensions

  • date (time)
  • sector (categorical)

Measures

  • active_accounts_value (sum)
  • txs_value (sum)
  • gas_used_sum_value (sum)
  • fee_native_sum_value (sum)
