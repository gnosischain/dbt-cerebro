int_execution_zodiac_module_proxies

Catalogue of every Zodiac module proxy ever deployed on Gnosis Chain. Decoded directly from the canonical Zodiac ModuleProxyFactory's ModuleProxyCreation(address indexed proxy, address indexed masterCopy) event. The factory address is hardcoded — there has only ever been one on Gnosis Chain at the canonical address. decode_logs handles its own start_month/end_month + incremental filter, so this model just sets the matching incremental_strategy switch and reshapes the decoded output. Do NOT add an outer block_timestamp filter. Foundation for downstream module-aware models — joined to int_execution_safes_module_events to identify which Zodiac module proxies are attached to which Safes (and to filter to a specific module type via the master_copy column).
