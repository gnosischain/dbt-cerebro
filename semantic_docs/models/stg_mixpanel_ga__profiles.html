stg_mixpanel_ga__profiles

One row per Mixpanel People/profile (latest snapshot), privacy-safe. Collapses the daily full-snapshot source (mixpanel_ga.mixpanel_raw_profiles) to the newest row per distinct_id. `pay` (the profile's Gnosis Pay Safe address) is pseudonymized with the same salted macro used on the on-chain gp_safe, so pay_safe_pseudonym joins directly to pseudonymize_address(gp_safe); 0 means no Gnosis Pay Safe set. initial_utm_* are Mixpanel's set-once first-touch campaign values (present on profiles even though absent from the event stream). No raw address or distinct_id is emitted.
