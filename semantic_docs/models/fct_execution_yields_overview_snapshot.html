fct_execution_yields_overview_snapshot

Long-format KPI snapshot table for the yields overview dashboard. One row per metric, with the latest value, 7-day change percent, and a display label (e.g. the best-APR pool name or best-APY token). Sourced from fct_execution_pools_daily (LP TVL/best APR), int_execution_lending_aave_daily (lending best APY/lender count), and the Savings xDAI rate model (sDAI APY/supply). Materialized as a small ReplacingMergeTree keyed by metric so the overview API views can do constant-time WHERE filters.

valueFloat64Current value for the metric (units vary by metric — USD for TVL/supply, percent for APR/APY, count for lenders).
