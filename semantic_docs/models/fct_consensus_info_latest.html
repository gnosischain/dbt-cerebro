fct_consensus_info_latest

The fct_consensus_info_latest model aggregates the most recent and 7-day summary metrics related to deposits, withdrawals, validator APY, validator status, and staked balances, providing a comprehensive snapshot of consensus-related financial activity and validator health.

Module: consensus

Status: candidate

Owner: analytics_team

Path: models/consensus/marts/fct_consensus_info_latest.sql

Tags: production consensus info

Columns

NameTypeDescription
change_pctFloat64Percentage change over 7 days; null if previous data is zero.
cntUInt64The count of deposit or withdrawal transactions, representing the number of events.
labelStringA descriptive identifier for the metric or category, such as deposits count or validator status.
total_amountFloat64The total monetary value of deposits or withdrawals, rounded to two decimal places, measured in the relevant currency units.
valueFloat64Numeric value of the metric; units vary (counts, amounts, percentages).
