int_execution_transfers_whitelisted_daily

The `int_execution_transfers_whitelisted_daily` model aggregates daily whitelisted token transfer data, including deposits, withdrawals, and transfers involving specific tokens, to support analysis of token movement activity within a defined time range.

Module: execution

Status: candidate

Owner: analytics_team
