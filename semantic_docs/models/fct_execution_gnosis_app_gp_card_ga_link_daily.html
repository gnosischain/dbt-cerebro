fct_execution_gnosis_app_gp_card_ga_link_daily

Daily + cumulative count of Gnosis Pay cards LINKED to a Gnosis App account, split by link_source. Backing fact for api_execution_gnosis_app_gp_card_ga_link_daily. Module-agnostic recovery series that resumes past the June-2026 plateau of the DelayModule-only fct_execution_gnosis_app_gpay_wallets_daily. Each card carries exactly one (highest-precedence) link_source upstream, so summing across sources equals total distinct GA-linked cards. This is a LINK (a GA account controls/funds the card), NOT the onboarded_via_ga-vs-imported class (unrecoverable post-migration). Aggregate counts only — no address/pseudonym.
