int_consensus_blocks_daily

The int_consensus_blocks_daily model aggregates daily consensus block production metrics, including counts of blocks produced, blob commitments, and missed blocks, to monitor network performance over time.

Module: consensus

Status: candidate

Owner: analytics_team

Path: models/consensus/intermediate/int_consensus_blocks_daily.sql

Tags: production consensus blocks

Columns

NameTypeDescription
blocks_missedUInt64Number of blocks missed on the given date.
blocks_producedUInt64Number of blocks produced on the given date.
dateDateThe day for which the summary is computed, in UTC.
genesis_time_unixUInt64The Unix timestamp indicating the network's genesis (start) time, used as a reference point for time calculations.
seconds_per_slotUInt64The duration of each consensus slot in seconds, used to determine block timing and missed blocks.

Dimensions
