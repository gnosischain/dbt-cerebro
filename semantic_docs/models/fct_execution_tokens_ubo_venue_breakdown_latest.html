fct_execution_tokens_ubo_venue_breakdown_latest

Latest-snapshot split of each token's positive-balance supply across UBO venues. One row per (token_address, venue): protocol rows aggregate fct_ubo_supply_claims_resolved_daily by protocol on the latest complete date (< today) in int_execution_tokens_balances_daily, and a residual 'direct' row carries the token's total balance (zero address excluded) minus all protocol-held balances, floored at zero. percentage expresses each venue's share of the token's total balance on that date.
