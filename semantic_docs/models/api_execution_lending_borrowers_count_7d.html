api_execution_lending_borrowers_count_7d

KPI view of currently-active borrowers on Gnosis (Aave V3, SparkLend). Mirror of api_execution_lending_lenders_count_7d but for borrowers. "Active borrowers" is a STOCK measure — unique wallets currently holding a positive debt position. One row per protocol plus an 'ALL'-protocols aggregate. change_pct compares against the count 7 days earlier.
