contracts_BalancerV3_Vault_events

Decoded event log stream from the Balancer V3 Vault contract on Gnosis Chain. One row per emitted log; decoded_params is a Map(String, String) with event-specific payload (includes pool address, amounts arrays, swap-fee amount arrays, and wrap/unwrap fields). Produced via the standard decode_logs macro. Backfilled from 2024-01-01 in 6-month batches.
