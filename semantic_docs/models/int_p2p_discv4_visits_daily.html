int_p2p_discv4_visits_daily

The `int_p2p_discv4_visits_daily` model aggregates daily peer-to-peer DISCv4 visit data, providing insights into visit volume and success rates for network analysis and monitoring.

Module: p2p

Status: candidate

Owner: analytics_team

Path: models/p2p/intermediate/int_p2p_discv4_visits_daily.sql

Tags: production p2p discv4

Columns

NameTypeDescription
crawlsUInt64The number of unique crawl sessions initiated on the given date.
dateDateTime64The specific day for which the visit data is aggregated, representing the start of the day.
successful_visitsUInt64The count of visits that either had no dial errors or where crawl errors are null, indicating successful connection attempts.
total_visitsUInt64The total number of visits that ended on the given date.
