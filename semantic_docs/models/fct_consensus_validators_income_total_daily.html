fct_consensus_validators_income_total_daily

Network-wide daily consensus income in GNO, summed across every validator (including exited / zero-balance). Derived from int_consensus_validators_income_daily, which spec-bounds income per-validator (see the block comment at the top of that model for the beacon reward-cap derivation), so no aggregate-level outlier filter is required — the input is already clean. Materialized as a physical table (not a view) so the dashboard API returns in milliseconds: the source fact has 200M+ rows and running SUM GROUP BY on every request would 30s-timeout Vercel functions. Rebuild cost is trivial (~1568 output rows) and hooks into the dbt DAG so each upstream rebuild refreshes this table too.
