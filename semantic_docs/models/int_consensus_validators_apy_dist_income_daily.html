int_consensus_validators_apy_dist_income_daily

Network-wide daily APY quantile distribution, built on int_consensus_validators_income_daily (the spec-bounded APY source). Parallel to the frozen int_consensus_validators_dists_daily lineage which reads per_index_apy_daily; both coexist. This model feeds the new api_consensus_validators_apy_dist_income_daily view consumed by the "Network Average APY" band chart on the Consensus tab. Filter: apy BETWEEN 0 AND 200 (drops exited / numerical noise) AND effective_balance_gno > 0 (drops pending / exited). Emits T-Digest quantiles q05..q95, balance-weighted mean avg_apy_weighted matching fct_consensus_validators_apy_mean_daily, and validators_included for sanity audit.
