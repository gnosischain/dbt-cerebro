int_p2p_discv5_visits_daily

The `int_p2p_discv5_visits_daily` model aggregates daily peer-to-peer network visit data, focusing on successful and total visits associated with specific protocol forks for analytical insights.

Module: p2p

Status: candidate

Owner: analytics_team

Path: models/p2p/intermediate/int_p2p_discv5_visits_daily.sql

Tags: production p2p discv5

Columns

NameTypeDescription
cl_fork_nameStringThe human-readable name corresponding to the fork digest, indicating the protocol version or upgrade.
dateDateTimeThe day for which the visit data is aggregated, in UTC.
fork_digestStringA unique identifier (hash) representing a specific protocol fork in the network.
fork_nameStringName of the fork associated with the peer, e.g., 'Phase0', 'Altair'.
successful_visitsUInt64Number of visits that were successful, excluding errors or failed attempts.
total_visitsUInt64Total number of visits recorded on the given day.

Dimensions

  • date (time)
  • fork_name (categorical)

Measures

  • successful_visits_value (sum)
