contracts_wxdai_calls

Decoded transaction-level calls to the WXDAI contract on Gnosis. Thin wrapper around the decode_calls macro; one row per transaction calling the contract.

Module: contracts

Status: docs_only

Owner: analytics_team

Path: models/contracts/tokens/contracts_wxdai_calls.sql
