contracts_wxdai_calls

The `contracts_wxdai_calls` model captures and decodes call interactions with the specified wxDai contract, providing detailed insights into transaction activity over time.

Module: contracts

Status: docs_only

Owner: analytics_team

Path: models/contracts/tokens/contracts_wxdai_calls.sql
