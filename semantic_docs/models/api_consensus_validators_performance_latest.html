api_consensus_validators_performance_latest

Public API view — one row per validator with latest snapshot fields plus 30-day and lifetime aggregates.

Module: consensus

Status: approved

Owner: analytics_team

Path: models/consensus/marts/api_consensus_validators_performance_latest.sql

Tags: production consensus tier1 api:validators_performance granularity:latest

Columns

NameTypeDescription
as_of_dateDateDate the snapshot is computed as of (max date in the underlying data).
balance_gnoFloat64Validator balance at the latest date, in GNO.
consensus_income_amount_30d_gnoFloat64Sum of consensus income over the trailing 30 days for the validator, in GNO.
effective_balance_gnoFloat64End-of-day effective balance in real GNO (the 1-GNO-step stake used in the reward-cap formula; the spec's 32-mGNO step equals 1 GNO).
latest_dateDateTime64Date of the validator's most recent income row (max date in int_consensus_validators_income_daily).
proposed_blocks_count_lifetimeUInt64Total number of blocks proposed by the validator over its lifetime.
proposer_reward_total_30d_gnoFloat64Sum of proposer rewards over the trailing 30 days for the validator, in GNO.
pubkeyStringLowercased validator public key.
total_income_estimated_gnoFloat64Lifetime estimated consensus income for the validator, in GNO.
validator_indexUInt64Beacon-chain validator index.
withdrawal_addressNullable(String)20-byte withdrawal address derived from credentials when prefix is 0x01 or 0x02; NULL otherwise.
withdrawal_credentialsStringLowercased withdrawal credentials.
