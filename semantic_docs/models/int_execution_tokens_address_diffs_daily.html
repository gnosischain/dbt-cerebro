int_execution_tokens_address_diffs_daily

This model calculates daily net address-level token transfer deltas, capturing inflows and outflows for each address to support analysis of token movement patterns and address activity over time.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/tokens/intermediate/int_execution_tokens_address_diffs_daily.sql

Tags: production execution tokens address_deltas

Columns

NameTypeDescription
ASThe recipient address in the transfer, stored in lowercase, representing the address receiving tokens.
addressStringAccount address whose balance changed.
dateDateThe specific calendar date for the recorded token transfer net delta.
net_deltaFloat64Net change in the account balance.
symbolStringThe token's symbol, representing its identifier in a human-readable format.
token_addressStringThe lowercase hexadecimal address of the token involved in the transfer.
token_classStringClass of the token.
