int_execution_gpay_wallet_owners

Current owner snapshot for Gnosis Pay Safes. Thin filter over int_execution_safes_current_owners (the generic Safe ownership model) so post-setup owner mutations (AddedOwner / RemovedOwner / ChangedThreshold) are reflected, not just the creation-time SafeSetup snapshot. One row per (pay_wallet, owner) pair.

ownerStringA current owner of the pay wallet (lowercase, 0x-prefixed).
pay_walletStringThe Gnosis Pay wallet (safe) address.
