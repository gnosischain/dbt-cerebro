stg_crawlers_data__cow_api_trade_fees

Staging view over cow_api_trade_fees (ingested by click-runner from the CoW Protocol API). Exposes protocol fees per fill for trades that use the surplus-based fee model (2024+, and partially mid-2023 onwards). One row per (order_uid, tx_hash, log_index): partially-fillable orders settle across multiple trades sharing an order_uid, each fill with its own executedProtocolFees — downstream joins must use the full per-fill key.
