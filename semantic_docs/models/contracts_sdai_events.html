contracts_sdai_events

The `contracts_sdai_events` model captures and processes event logs related to the specified smart contract, enabling analysis of contract interactions and activities over time.

Module: contracts

Status: docs_only

Owner: analytics_team

Path: models/contracts/tokens/contracts_sdai_events.sql

Tags: production contracts sdai events

Columns

NameTypeDescription
amountStringThe amount transferred, in wei or relevant units.
block_numberUInt64The number of the block containing the event.
block_timestampDateTimeThe timestamp when the block was mined, in UTC.
contract_addressStringThe address of the contract emitting the event.
event_dataJSONThe JSON payload containing event-specific data.
event_nameStringThe name of the event emitted.
event_signatureStringThe signature of the event, used for identification.
log_indexUInt64The index of the log within the block, used to order logs.
recipient_addressStringThe address receiving tokens or assets, if applicable.
sender_addressStringThe address initiating the transaction.
transaction_hashStringThe hash of the transaction associated with the event.
transaction_indexUInt64The index of the transaction within the block.
