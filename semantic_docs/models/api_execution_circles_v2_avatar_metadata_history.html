api_execution_circles_v2_avatar_metadata_history

Historical timeline of every Circles v2 avatar metadata change. Thin passthrough over int_execution_circles_v2_avatar_metadata_history with one row per (avatar, metadata_digest) ever announced, in event order, carrying valid_from / valid_to / is_current and the parsed IPFS profile fields (name, symbol, description, imageUrl, previewImageUrl) so the dashboard can render the full sequence of name + image updates per avatar.
