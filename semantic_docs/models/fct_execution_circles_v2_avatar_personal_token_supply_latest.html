fct_execution_circles_v2_avatar_personal_token_supply_latest

One-row-per-avatar summary of a Circles v2 avatar's own personal CRC token: total circulating supply, how much is wrapped as ERC-20, and the wrapped share. Rolled up from fct_execution_circles_v2_avatar_token_distribution so the supply respects the same 0.001 CRC dust threshold used by the distribution fact. The matching api_ is a thin passthrough.

Columns

NameTypeDescription
avatarStringThe avatar whose personal CRC token is being summarised.

wrapped_demurragedFloat64Demurrage-adjusted equivalent of wrapped.
wrapped_pctNullable(Float64)wrapped / supply × 100, rounded to 1 decimal. NULL when supply = 0.
