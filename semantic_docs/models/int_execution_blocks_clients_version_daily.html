int_execution_blocks_clients_version_daily

The model aggregates daily counts of execution blocks grouped by client and version, supporting incremental updates for tracking client activity over time.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/blocks/intermediate/int_execution_blocks_clients_version_daily.sql

Tags: production execution blocks

Columns

NameTypeDescription
clientStringThe name of the client associated with the execution blocks, with certain values mapped to 'Unknown' for consistency.
cntUInt64The total number of execution blocks recorded for the given client and version on that date.
dateDateTime64The specific day when the execution blocks were recorded, truncated to the start of the day.
versionStringThe client software version extracted from decoded extra data, indicating the version used during execution.

  • date (time)
