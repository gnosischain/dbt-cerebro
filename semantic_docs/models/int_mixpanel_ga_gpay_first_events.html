int_mixpanel_ga_gpay_first_events

INTERNAL ONLY. One row per Gnosis Pay account (identity_role=initial_owner) per first-event type, with the account's Mixpanel UTM attribution attached via user_pseudonym == user_id_hash. event_type is 'funded' (first gpay_funded inflow) or 'first_payment' (first gpay_payment). Accounts with no Mixpanel match get 'unknown' campaigns so totals reconcile to the full GP funnel. Carries user_pseudonym — never exposed to cerebro-api or MCP.
