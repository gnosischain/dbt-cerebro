fct_execution_transactions_snapshots

This table provides snapshot metrics of execution transactions, including counts, fees, and active account counts over various time windows for business analysis and trend monitoring.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/transactions/marts/fct_execution_transactions_snapshots.sql

Tags: production execution transactions

Columns

NameTypeDescription
change_pctFloat64Percentage change compared to previous window; null if not applicable.
labelStringLabel categorizing the metric, e.g., 'Transactions', 'FeesNative', 'ActiveAccounts'.
max_dateDateThe most recent date available in the source transaction data, used as the reference point for calculating time window bounds.
valueFloat64Numeric value of the metric for the given window.
windowStringTime window identifier, e.g., '1D', '7D', '30D', '90D', or 'All'.

Dimensions

  • max_date (time)
  • label (categorical)
  • window (categorical)

Measures

  • value_value (sum)
  • change_pct_value (average)
