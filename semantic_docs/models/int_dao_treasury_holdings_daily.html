int_dao_treasury_holdings_daily

Daily GnosisDAO holdings on Gnosis Chain. Unions ERC-20 token balances (from int_execution_tokens_balances_daily) with lending supply positions (from int_execution_lending_aave_user_balances_daily) for the wallets listed in the dao_treasury_wallets seed. Each row is a (date, wallet, token, position_type) combination with USD valuation.

NameTypeDescription
balanceFloat64Token balance in native units.
balance_usdNullable(Float64)Balance in USD using daily token price. Null when no price available.

position_typeString'wallet' for token balances, 'lending' for Aave/SparkLend supply.
protocolStringLending protocol name (Aave V3, SparkLend) or empty for wallet holdings.

token_classStringToken classification. For wallet holdings this comes from tokens_whitelist; for lending positions it is set to 'LENDING'.
wallet_addressNullable(String)Gnosis Chain address of the DAO wallet.
wallet_labelStringHuman-readable wallet label from dao_treasury_wallets seed.
