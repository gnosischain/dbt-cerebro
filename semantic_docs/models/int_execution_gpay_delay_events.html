int_execution_gpay_delay_events

Decoded events emitted by the Zodiac Delay Module proxies attached to GP Safes. Includes: DelaySetup — module enabled and bound to a Safe TransactionAdded — admin action queued by the original setup owner (the privacy-respecting "user did something admin-y" signal — single most useful Delay event for analytics) TxCooldownSet — cooldown configuration mutation TxExpirationSet — expiration configuration mutation TxNonceSet — nonce reset decode_logs filters logs.address to (SELECT address FROM contracts_gpay_modules_registry WHERE contract_type='DelayModule') and joins event_signatures on the delayModMastercopy address — that's the abi_source_address column the registry exposes.
