int_execution_gpay_activity

Incremental model that captures individual Gnosis Pay wallet transactions including payments, deposits, withdrawals, and cashback events with USD valuations.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/gpay/intermediate/int_execution_gpay_activity.sql

Tags: production execution gpay activity

Columns

NameTypeDescription
actionStringThe type of activity (e.g., Payment, Deposit, Withdrawal, Cashback).
amountFloat64The decimal-adjusted token amount.
amount_usdFloat64The USD-equivalent value of the transaction.
block_timestampDateTime64(0, 'UTC')The UTC timestamp of the block containing the transaction.
counterpartyStringThe counterparty address in the transaction.
dateDateThe calendar date of the transaction.
directionStringThe direction of the transaction (e.g., inbound, outbound).
symbolStringThe token symbol involved in the transaction.
token_addressStringThe contract address of the token.
transaction_hashStringThe unique hash identifier for the transaction.
value_rawFloat64The raw token amount before decimal adjustment.
wallet_addressStringThe Gnosis Pay wallet address involved in the transaction.
