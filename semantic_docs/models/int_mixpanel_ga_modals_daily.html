int_mixpanel_ga_modals_daily

Daily open counts for each modal/bottom-sheet component in Gnosis App. Sourced from events where event_category = 'modal' (Open Modal or Open Bottom Sheet) with a non-empty bottom_sheet property. Known bottom_sheet values: PasskeyStepsModal (passkey onboarding steps), ModalJoin (join Circles), SelectSwapAsset (swap token picker), ModalReceive (receive funds QR), ModalSend (send funds flow), CardActivationStepsModal (Gnosis Pay card activation). Production-only (app.gnosis.io).
