int_execution_circles_v2_balances_daily

Daily end-of-day token balances per account for Circles v2 tokens with demurrage adjustment.

Module: execution

Status: docs_only

Owner: analytics_team

Path: models/execution/Circles/intermediate/int_execution_circles_v2_balances_daily.sql

Tags: production execution circles_v2 balances

Columns

NameTypeDescription
accountStringAddress of the token holder.
balance_rawInt256Cumulative nominal balance in native token units. Hub balances and demurrage wrappers are in demurrage units; inflationary wrappers are in static units.
circles_typeUInt8Circles representation enum for the position (0 = Demurrage, 1 = Inflation).
dateDateCalendar date of the balance snapshot.
demurraged_balance_rawInt256Balance converted to demurrage units at the snapshot timestamp. Inflationary wrappers use the snapshot day as the conversion time basis; demurrage-native positions use the last activity timestamp.
last_activity_tsUInt64Unix timestamp of the most recent transfer activity for this position.
snapshot_tsUInt64Unix timestamp used for the end-of-day snapshot (end of day or chain now for today).
token_addressStringAddress of the token contract.
