fct_execution_gnosis_app_swaps_daily

Daily Gnosis App swap activity, derived from int_execution_gnosis_app_swaps. Backing fact for api_execution_gnosis_app_swaps_daily. Dune-parity semantic: one swap = one PreSignature event (= one signed order intent) via the Cometh bundler. Orders pre-signed but never filled are still counted as n_swaps; volume_usd reflects only filled orders.
