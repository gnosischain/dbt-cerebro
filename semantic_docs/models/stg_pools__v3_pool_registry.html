stg_pools__v3_pool_registry

Complete registry of Uniswap V3 and Swapr V3 (Algebra) pools on Gnosis Chain. Combines factory creation events with pool Initialize events to produce a single source of truth for pool metadata. Includes fee tier (static for UniV3, NULL for Swapr with dynamic fees), tick spacing, initialization price/tick, and creation timestamp. Replaces the former int_execution_yields_v3_pool_meta view.
