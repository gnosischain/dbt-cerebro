api_p2p_topology_latest

The api_p2p_topology_latest model provides a real-time view of peer-to-peer network topology, capturing the latest connections and their geographic and organizational attributes for analysis and monitoring.

Module: p2p

Status: candidate

Owner: analytics_team

Path: models/p2p/marts/api_p2p_topology_latest.sql

Tags: production p2p tier0 api:network_topology granularity:latest

Columns

NameTypeDescription
cntUInt64Count of occurrences or connections between the peer and neighbor, representing the connection strength or frequency.
dateDateThe date when the topology data was recorded, in YYYY-MM-DD format.
neighbor_cityStringCity where the neighboring peer is located.
neighbor_clientStringClient software or implementation used by the neighboring peer.
neighbor_countryStringCountry where the neighboring peer is located.
neighbor_discovery_id_prefixUInt64Prefix of the unique identifier for the neighboring peer.
neighbor_latFloat64Latitude coordinate of the neighboring peer's location.
neighbor_lonFloat64Longitude coordinate of the neighboring peer's location.
neighbor_orgStringOrganization associated with the neighboring peer, if available.
peer_cityStringCity where the peer is located.
peer_clientStringThe client software or implementation used by the peer.
peer_countryStringCountry where the peer is located.
peer_discovery_id_prefixUInt64Prefix of the unique identifier for the peer discovered in the network.
peer_latFloat64Latitude coordinate of the peer's geographic location.
peer_lonFloat64Longitude coordinate of the peer's geographic location.
peer_orgStringOrganization associated with the peer, if available.
protocolStringThe communication protocol used between peers, such as TCP or UDP.
