fct_execution_cow_trades

Canonical CoW Protocol trades table. Joins on-chain trade data from int_execution_cow_trades with off-chain protocol fee data from stg_crawlers_data__cow_api_trade_fees (ingested from the CoW API). fee_usd logic: pre-2024 CoW used an explicit fee model (fee_amount_raw > 0 in the Trade event); 2024+ CoW switched to surplus-based fees (not visible in events) sourced from executedProtocolFees via /api/v2/trades; the transition period (mid-2023 through early 2024) is mixed — COALESCE picks whichever source is available per trade. fee_source identifies where the fee came from ('onchain', 'api', or NULL). NOTE — ClickHouse LEFT JOIN with join_use_nulls=0 (default) fills unmatched String columns with '' instead of NULL, so we check fee_token != '' rather than IS NOT NULL to correctly detect a real API match.
