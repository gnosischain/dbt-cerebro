int_execution_circles_v2_inviter_fees

Per-event Circles v2 inviter-fee transfers — wrapped-CRC ERC-20 Transfer events in the same tx as a Hub.PersonalMint, from invitee to inviter per the avatar→inviter mapping in int_execution_circles_v2_avatars. Used by the WEAU weekly-earners definition. The >= 1 CRC threshold is applied downstream so this stays event-grain and incremental-safe.
