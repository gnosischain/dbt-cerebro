contracts_backedfi_bCSPX_Oracle_events

The `contracts_backedfi_bCSPX_Oracle_events` model captures and processes blockchain event logs related to the BackedFi bCSPX Oracle contract, enabling analysis of oracle interactions and event occurrences over time.
