int_mixpanel_ga_user_lifecycle

All-time per-user lifecycle metrics for Gnosis App (app.gnosis.io) production traffic. One row per user_id_hash. Full rebuild on every run (materialized=table); lifetime aggregates cannot be computed incrementally without reading all history anyway. Captures: identification status (wallet vs anonymous), active lifespan, feature engagement (passkey logins, Circles mints, marketplace purchases, swap actions), and event category breakdowns. is_identified=1 means the user connected a wallet at least once (Mixpanel distinct_id is a wallet address, not a $device:uuid).
