fct_execution_lending_weekly

Weekly fact aggregating lending activity per (week, protocol, token) across Aave V3 and SparkLend on Gnosis. Built from int_execution_lending_aave_daily by collapsing days into ISO weeks (toStartOfWeek). APYs use the end-of-week value (argMax by date); volumes are summed across the week; distinct lender/borrower counts come from the merged daily bitmap states. Excludes the current (incomplete) week.
