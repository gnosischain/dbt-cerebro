int_mixpanel_ga_gnosis_app_first_events

INTERNAL ONLY. One row per Gnosis App account (user_pseudonym) per first-conversion type, with the account's Mixpanel UTM attribution attached via user_pseudonym == user_id_hash. conversion_kind is one of topup / swap_filled / token_offer_claim / marketplace_buy. Unlike the GP side, App conversions match the Mixpanel identified-user set 70-83%. Accounts with no Mixpanel match get 'unknown' campaigns. Carries user_pseudonym — never exposed to cerebro-api or MCP.
