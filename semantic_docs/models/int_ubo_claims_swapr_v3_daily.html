int_ubo_claims_swapr_v3_daily

Per-user UBO supply claims for Swapr V3 (Algebra). Standardized shape: one row per (date, container_address, ubo_address, token_address) capturing each LP's net withdrawable token balance in a Swapr V3 pool. Same two-track approach as Uniswap V3, but simpler because Swapr's IncreaseLiquidity event includes the pool address directly (no transaction_hash join needed). Each Swapr V3 pool is its own token custodian so container_address is the individual pool address. Track A (direct LPs): the pool Mint/Burn owner field is the real LP. Track B (via NPM): the owner is the NonfungiblePositionManager (0x91fd...). The real LP is resolved from NPM Transfer events.
