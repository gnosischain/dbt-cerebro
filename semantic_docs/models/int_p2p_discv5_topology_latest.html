int_p2p_discv5_topology_latest

This model captures the latest peer-to-peer discovery topology, including peer and neighbor information along with associated metadata and geographical details, to analyze network connectivity patterns.

Module: p2p

Status: candidate

Owner: analytics_team

Path: models/p2p/intermediate/int_p2p_discv5_topology_latest.sql
