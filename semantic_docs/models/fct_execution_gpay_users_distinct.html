fct_execution_gpay_users_distinct

Deduplicated Gnosis Pay user projection for the semantic layer's cross-sector user-overlap path. One row per user_pseudonym, collapsed from int_execution_gpay_safe_identities (1 row per user_pseudonym x identity_role x gp_safe), with role-presence flags (has_initial_owner, has_delegate, has_safe_self) and distinct safe/role counts. Kept at 1-row-per-user grain so user-keyed semantic metrics can use a plain count aggregation instead of count_distinct, which the semantic-layer planner emits as a raw function name ClickHouse does not expose.
