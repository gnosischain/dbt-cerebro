int_revenue_holdings_fees_daily

Daily per-user imputed holdings fees for EURe, USDC.e, BRLA, ZCHF. `fees = balance_usd * daily_rate_for_symbol`, where daily rates are annualised APYs expressed per-day compounded ((1 + daily_rate)^365 - 1 ≈ APY) and declared as Jinja constants at the top of the SQL (edit to retune). Symbol buckets (sum-into-canonical symbol): - EURe: native EURe + EURe held in Aave V3 (aGnoEURe) - USDC.e: native USDC.e + USDC.e held in Aave V3 (aGnoUSDCe) - BRLA: native only - ZCHF: ZCHF + svZCHF (no lending market) Sources: - int_execution_tokens_balances_daily — native ERC20 balances - int_execution_lending_aave_user_balances_daily — Aave V3 aToken balances; rows already carry the underlying reserve symbol (e.g. 'EURe' for an aGnoEURe holder), not the aToken symbol. SparkLend is excluded.
