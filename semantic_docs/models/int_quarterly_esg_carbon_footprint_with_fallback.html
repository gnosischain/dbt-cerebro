int_quarterly_esg_carbon_footprint_with_fallback

Extends fct_esg_carbon_footprint_uncertainty with estimated daily CO2 and energy values for periods where Ember carbon intensity data is missing. Uses production fact table values for historical dates and recomputes estimates from node distribution, client efficiency, and forward-filled carbon intensity for the gap period.
