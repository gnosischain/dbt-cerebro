api_execution_tokens_volume_daily

The api_execution_tokens_volume_daily model provides daily aggregated data on the trading volume of different tokens in both native token units and USD, supporting analysis of token liquidity and trading activity over time.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/tokens/marts/api_execution_tokens_volume_daily.sql

Tags: production execution tier1 api:tokens_volume granularity:daily

Columns

NameTypeDescription
dateDateThe specific calendar date for the recorded trading volume data.
tokenStringThe identifier of the token for which the volume is reported.
token_classStringThe classification category of the token, such as utility or security token.
value_nativeFloat64The total trading volume for the token on the given date in native token units.
value_usdFloat64The total USD trading volume for the token on the given date.

Dimensions

  • date (time)
  • token (categorical)
  • token_class (categorical)

Measures

  • value_native_value (sum)
  • value_usd_value (sum)
