contracts_aaveV3_PoolInstance_events

The `contracts_aaveV3_PoolInstance_events` model captures and processes event logs related to Aave V3 pool instances, enabling analysis of protocol interactions and state changes over time.

Module: contracts

Status: docs_only

Owner: analytics_team

Path: models/contracts/aave/contracts_aaveV3_PoolInstance_events.sql

Tags: production aave v3 contracts events

Columns

NameTypeDescription
amountStringAmount involved in the event, denominated in wei.
block_numberUInt64Number of the block containing the event.
block_timestampDateTimeTimestamp of the block when the event was emitted, in UTC.
event_typeStringType of the event emitted, e.g., 'Deposit', 'Withdraw'.
log_indexUInt64Index of the log within the block, used to order events.
pool_addressStringAddress of the Aave V3 Pool contract instance.
reserve_assetStringAddress of the reserve asset involved in the event.
transaction_hashStringUnique hash identifying the transaction that emitted the event.
user_addressStringAddress of the user involved in the event.
