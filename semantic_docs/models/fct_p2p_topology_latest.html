fct_p2p_topology_latest

This view consolidates the latest peer-to-peer topology data from DiscV4 and DiscV5 protocols, enabling analysis of network connections and geographic distribution.

Module: p2p

Status: candidate

Owner: analytics_team

Path: models/p2p/marts/fct_p2p_topology_latest.sql

Tags: production p2p topology

Columns

NameTypeDescription
cntUInt64Number of observed connections between the peer and neighbor nodes.
dateDateThe date when the topology data was recorded.
neighbor_cityStringCity where the neighbor node is located.
neighbor_clientStringClient software used by the neighbor node.
neighbor_countryStringCountry where the neighbor node is located.
neighbor_discovery_id_prefixUInt64Prefix of the neighbor's discovery identifier.
neighbor_ipStringIP address of the neighboring node in the topology.
neighbor_latFloat64Latitude coordinate of the neighbor node, in decimal degrees.
neighbor_lonFloat64Longitude coordinate of the neighbor node, in decimal degrees.
neighbor_orgStringOrganization associated with the neighbor node.
peer_cityStringCity where the peer node is located.
peer_clientStringClient software used by the peer node.
peer_countryStringCountry where the peer node is located.
peer_discovery_id_prefixUInt64Prefix of the peer's discovery identifier.
peer_hostnameStringHostname of the peer node.
peer_ipStringIP address of the peer node initiating the connection.
peer_latFloat64Latitude coordinate of the peer node, in decimal degrees.
peer_lonFloat64Longitude coordinate of the peer node, in decimal degrees.
peer_orgStringOrganization associated with the peer node.
