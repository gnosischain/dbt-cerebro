fct_execution_gnosis_app_users_daily

Daily user counts. new_users = distinct addresses whose first-ever activity hit was on this date; active_users = distinct addresses with any non-onboard activity; returning_users = active today AND had any activity in the prior 7 days; reactivated_users = active today, inactive for prior 30 days, but active before that. Dense calendar spine so cumulative_users is always continuous.
