int_execution_transactions_by_project_hourly_recent

This model aggregates hourly execution transaction data by project for the recent two-day period, providing insights into transaction volume, active accounts, and associated fees for business analysis.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/transactions/intermediate/int_execution_transactions_by_project_hourly_recent.sql

Tags: production execution transactions hourly

Columns

NameTypeDescription
active_accountsUInt64Count of distinct active sender addresses during the hour.
addressStringThe blockchain address associated with a transaction, used for label mapping.
fee_native_sumFloat64Sum of gas used multiplied by gas price, converted to native units (e.g., ETH).
hourDateTimeThe start of the hour period for the aggregated data.
projectStringThe name of the project associated with the transaction, or 'Unknown' if not labeled.
sectorStringThe sector classification of the project, or 'Unknown' if not labeled.
tx_countUInt64Total number of transactions in the hour for the project.
ua_bitmap_stateStringBitmap state representing unique active addresses for the project.
