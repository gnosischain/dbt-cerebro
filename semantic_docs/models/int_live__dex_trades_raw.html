int_live__dex_trades_raw

Cached, rolling 48h window of unified DEX swaps across all supported protocols (Uniswap V3, Swapr V3, Balancer V2, Balancer V3) on Gnosis Chain. Normalizes amounts by token decimals, resolves symbols, and attaches USD prices per side via ASOF join against daily token prices. The final `amount_usd` uses the LEAST of the two priced sides when both are known (a conservative notional that rejects inflated long-tail prices), or the single available price when only one side is priced. MATERIALIZATION: incremental ReplacingMergeTree, refreshed by a k8s CronJob running `dbt run --select +tag:live` every ~5 min. First run backfills the last 48h (source TTL). Each subsequent run delete+inserts the last `live_trades_overlap_minutes` (default 120 = 2h) to self-heal against cryo-live's out-of-order / bulk-attach inserts. 48h TTL on `block_timestamp` prevents unbounded growth and matches the source. Downstream api views read this table directly, so dashboard queries never trigger decode-log scans over `execution_live.logs`.
