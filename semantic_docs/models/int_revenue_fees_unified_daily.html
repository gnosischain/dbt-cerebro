int_revenue_fees_unified_daily

Unified daily per-user revenue fees across all streams. UNION ALL of the three per-stream daily fee intermediates with a stream_type label ('holdings'|'sdai'|'gpay'). Used by cross-stream active-user cohort marts where a user must not be double-counted across holdings / sDAI / gpay streams. Materialized as a view.
