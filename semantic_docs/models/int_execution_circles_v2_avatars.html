int_execution_circles_v2_avatars

This model aggregates daily counts of avatar registrations by type (Human, Group, Organization) to monitor user onboarding activities within the platform.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/Circles/intermediate/int_execution_circles_v2_avatars.sql

Tags: production execution circles avatars

Columns

NameTypeDescription
avatar_typeStringThe category of the registered avatar, classified as Human, Group, or Organization.
block_timestampDateTimeUTC timestamp of the block containing the event.
cntUInt64The total number of registration events for the given avatar type on the specified date.
dateDateTime64The date (at day granularity) when the registration events occurred.
event_nameStringName of the registration event ('RegisterHuman', 'RegisterGroup', 'RegisterOrganization').
from_addressStringAddress that initiated the registration.
registration_event_idStringUnique identifier for the registration event.
source_tableStringSource table name for lineage tracking.
to_addressStringRecipient address associated with the registration.
transaction_hashStringHash of the transaction that triggered the event.
