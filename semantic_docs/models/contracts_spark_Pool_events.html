contracts_spark_Pool_events

The `contracts_spark_Pool_events` model captures and processes event logs from Spark Protocol's Pool contract, enabling analysis of protocol interactions, reserve data updates, and yield rate changes over time. This model is used to extract ReserveDataUpdated events containing liquidityRate for stablecoin yield calculations.

Module: contracts

Status: docs_only

Columns

NameTypeDescription
block_numberUInt64Number of the block containing the event.
block_timestampDateTime64Timestamp of the block when the event was emitted, in UTC.
contract_addressNullable(String)Address of the Spark Pool contract emitting the event.
decoded_paramsMap(String, Nullable(String))Map of decoded event parameters. For ReserveDataUpdated events, contains 'reserve', 'liquidityRate', 'stableBorrowRate', 'variableBorrowRate', 'liquidityIndex', and 'variableBorrowIndex'.
event_nameStringName of the event emitted, e.g., 'ReserveDataUpdated', 'Supply', 'Withdraw'.
log_indexUInt64Index of the log within the block, used to order events.
transaction_hashNullable(String)Unique hash identifying the transaction that emitted the event.
transaction_indexUInt64Index of the transaction within the block.
