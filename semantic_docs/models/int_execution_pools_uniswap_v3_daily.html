int_execution_pools_uniswap_v3_daily

Daily Uniswap V3 pool token balances with oracle price enrichment. Built from event deltas (Mint, Burn, Swap, Collect, Flash) with static fee rates from PoolCreated (fee_tier_ppm). Separates reserves from unclaimed fees: reserve_amount = Mint/Burn liquidity + swap deltas net of fees; fee_amount = swap fee income − collected fees + flash loan fees. Includes oracle price (Dune ASOF join), TVL in USD, and pool-implied token price.
