int_execution_tokens_balance_cohorts_daily

Daily token balance cohort distributions, segmenting holders into balance buckets.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/tokens/intermediate/int_execution_tokens_balance_cohorts_daily.sql

Tags: production execution tokens balance_cohorts_daily

Columns

NameTypeDescription
balance_bucketStringThe balance range bucket (e.g., '0-10', '10-100', '100-1k').
cohort_unitStringThe unit used for cohort bucketing ('usd' for balance_usd, 'native' for balance).
dateDateThe specific date for which the token balance data is recorded.
holders_in_bucketUInt64Count of unique holders within the balance bucket.
symbolStringThe uppercase symbol representing the token.
token_addressStringThe lowercase hexadecimal address of the token contract.
token_classStringThe classification category of the token.
value_native_in_bucketFloat64Total native token balance held within the balance bucket.
value_usd_in_bucketFloat64Total USD value held within the balance bucket.
