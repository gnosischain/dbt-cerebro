api_consensus_credentials_daily

The api_consensus_credentials_daily model provides daily aggregated counts and percentage shares of different credential types used in the API, supporting operational and strategic analysis.

Module: consensus

Status: candidate

Owner: analytics_team

Path: models/consensus/marts/api_consensus_credentials_daily.sql

Tags: production consensus tier1 api:credentials granularity:daily

Columns

NameTypeDescription
cntUInt64The total number of credentials of a specific type recorded on the given date.
credentials_typeStringThe category or type of credentials used in the API during the specified date.
dateDateThe specific calendar date for the recorded credential data.
pctUInt64The percentage share of each credential type relative to the total credentials on that date, rounded to two decimal places.

Dimensions

  • date (time)
  • credentials_type (categorical)

Measures

  • cnt_value (sum)
  • pct_value (average)
