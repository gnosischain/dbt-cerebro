contracts_safe_registry

Registry consumed by decode_logs(contract_address_ref=ref('contracts_safe_registry')). Maps each Safe proxy → the singleton whose ABI should be used to decode its events. Columns required by decode_logs: address — the emitting contract (proxy) contract_type — 'SafeProxy', so callers can pass contract_type_filter abi_source_address — the singleton; decode_logs joins event_signatures on (topic0, abi_source_address) to find the ABI is_dynamic — 1 = discovered on-chain (all Safe proxies are) start_blocktime — earliest block at which this proxy can emit events discovery_source — provenance tag Tiny model — one row per Safe (thousands to low millions). Materialized as a table so decode_logs's compile-time get_columns_in_relation() works the moment dbt runs the downstream events model.
