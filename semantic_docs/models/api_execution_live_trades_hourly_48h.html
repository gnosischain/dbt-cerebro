api_execution_live_trades_hourly_48h

Hourly USD volume over the live window (up to 48h), stacked by protocol. Feeds the "last 48h volume" chart on the Live Feed tab. Volume is summed per swap event (per hop), so multi-hop trades contribute to every protocol they touch — this is the standard DEX-activity view. Rows with unknown USD are excluded. Anchored on `max(block_timestamp)` in `int_live__dex_trades_raw` so the window stays aligned with the feed.
