int_execution_tokens_transfers_daily

The int_execution_tokens_transfers_daily model aggregates daily transfer metrics for tokens, providing insights into token transfer volumes, counts, and user activity for analytics and reporting purposes.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/tokens/intermediate/int_execution_tokens_transfers_daily.sql

Tags: production execution tokens transfers_daily

Columns

NameTypeDescription
ASThe alias used for the symbol column in the SQL query, representing the token's symbol.
active_sendersUInt64Number of unique senders.
dateDateThe date of the transfer data, representing the day for which metrics are aggregated.
symbolStringThe token's symbol, representing its identifier in the ecosystem.
token_addressStringThe lowercase blockchain address of the token involved in transfers.
token_classStringClass of the token.
transfer_countUInt64Number of transfers.
ua_bitmap_stateStringBitmap state of user agents.
unique_receiversUInt64Number of unique receivers.
volume_tokenFloat64Total volume of tokens transferred.
volume_usdFloat64Total volume in USD.
