int_execution_circles_v2_avatar_metadata_targets

Deterministic queue of every (avatar, metadata_digest) pair the Circles v2 NameRegistry has ever announced. Built as a view over contracts_circles_v2_NameRegistry_events filtered to UpdateMetadataDigest rows with a non-empty digest, deduplicated to one row per (avatar, digest) using the latest event for that pair. Each row carries the derived metadata_digest_hex, ipfs_cid_v0, gateway_url, and an is_current_avatar_metadata flag marking the most recent digest per avatar. Drives both the one-time IPFS backfill script (scripts/circles/backfill_avatar_metadata.py) and the nightly fetch_and_insert_circles_metadata macro that knows which IPFS payloads still need to be fetched.
