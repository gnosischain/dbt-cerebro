api_mixpanel_ga_traffic_daily

Not exposed via cerebro-api; accessible to the internal MCP / semantic layer. Daily Gnosis App traffic breakdown by HTTP referring domain. Sourced from int_mixpanel_ga_traffic_daily. referrer_domain is 'direct' when the Referer header is absent; initial_referrer_domain captures the first-ever referrer a user came from (Mixpanel super-property). Use case: marketing channel attribution and growth funnel inputs.
