int_execution_safes_current_owners

Current owner set per Safe. For each (safe_address, owner) pair, take the latest event by (block_number, log_index). If the latest event is safe_setup or added_owner, the owner is current; if removed_owner, the row is dropped. Threshold is the latest safe_setup / changed_threshold per Safe, denormalized onto every owner row. `became_owner_at` is the timestamp of the most recent event that added (or set up) this owner — for an owner who was removed and then re-added, it is the re-add time, not the original setup time. Cheap to fully rebuild every run; no incremental strategy needed.
