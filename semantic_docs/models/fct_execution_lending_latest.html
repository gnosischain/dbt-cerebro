fct_execution_lending_latest

Lender / borrower counts per (window, protocol, token) plus protocol-scoped 'ALL'-tokens rows and an all-protocols 'ALL' bucket for cross-protocol totals. Built off the daily lender/borrower bitmap states from int_execution_lending_aave_daily, merged via groupBitmapMerge over the last 7 full days for the current window, and the prior 7 days for change_pct. Widgets typically use `token = 'ALL'` rows and apply the optional `protocol` filter (filterField3). `protocol = 'ALL'` rows are retained for dashboards that intentionally aggregate across protocols (e.g. a "total lenders on Gnosis" KPI).
