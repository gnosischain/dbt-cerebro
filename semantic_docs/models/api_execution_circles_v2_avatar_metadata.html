api_execution_circles_v2_avatar_metadata

Per-avatar identity for Circles v2 — on-chain registration metadata (type, name, inviter, registration time) joined to the most recently fetched IPFS profile (display name, preview/image URL, description). Backs the "Avatar Identity" panel on the Circles dashboard.

Module: execution

Status: docs_only

Owner: analytics_team

Path: models/execution/Circles/marts/api_execution_circles_v2_avatar_metadata.sql

Tags: production execution tier1 api:circles_v2_avatar_metadata granularity:snapshot

Columns

NameTypeDescription
avatarStringAddress of the registered avatar.
avatar_typeStringAvatar category (Human, Group, Org).
current_gateway_urlNullable(String)Full IPFS gateway URL for the current digest using the configured `circles_ipfs_gateway` var.
current_ipfs_cid_v0Nullable(String)CIDv0 derived from `current_metadata_digest`.
current_metadata_digestNullable(String)0x-prefixed hex digest of the most recent metadata payload announced for this avatar by NameRegistry. NULL when the avatar has never published an IPFS profile.
invited_byStringAddress of the avatar that invited this avatar (NULL for Groups/Orgs).
metadata_descriptionString`description` field extracted from the current profile JSON. Populated for ~20% of avatars.
metadata_fetched_atNullable(DateTime)Timestamp the current IPFS payload was fetched from the gateway.
metadata_image_urlString`imageUrl` field from the current profile JSON. Rarely populated (~1.5%); prefer `metadata_preview_image_url` for thumbnails.
metadata_nameStringDisplay `name` field extracted from the current IPFS profile JSON. Populated for ~99.9% of resolved avatars.
metadata_preview_image_urlString`previewImageUrl` field from the current profile JSON. Populated for ~53% of avatars.
metadata_symbolString`symbol` field extracted from the current profile JSON. Rarely populated (group/org tokens only).
nameStringOn-chain name set at registration (NULL for Humans).
registered_atDateTime64UTC timestamp of the registration event.
token_idStringPersonal token contract address (NULL for Orgs).
