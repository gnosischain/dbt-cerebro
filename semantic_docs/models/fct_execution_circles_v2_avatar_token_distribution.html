fct_execution_circles_v2_avatar_token_distribution

Per-avatar distribution of holders of the avatar's personal CRC token, classified as Self / Wrapped (ERC-20) / Other Circles avatars / Other contracts. Materialised daily as the latest-snapshot backing table for api_execution_circles_v2_avatar_token_distribution (which is a thin passthrough). One row per (avatar, holder_category).
