fct_mixpanel_ga_campaign_funnel_weekly

Weekly growth funnel by UTM, first-touch attribution. Every funnel step in one tidy long model so "which campaign attracts users" reads end-to-end: signup -> card_ordered -> circles_created -> crc_minted -> funded -> first_payment -> starts_referring. step_order makes funnel-monotonicity checks easy; not every user passes every step, so cross-step comparisons are per-campaign trends, not a strict sequential funnel. K-anonymity: campaigns with < 5 all-time signups are bucketed into '_small_campaigns'. Aggregate-only output (no user_id_hash).
