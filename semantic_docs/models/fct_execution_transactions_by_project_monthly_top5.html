fct_execution_transactions_by_project_monthly_top5

This model aggregates and ranks execution transactions by project on a monthly basis, highlighting the top 5 projects per month based on transaction volume and related metrics for business analysis.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/transactions/marts/fct_execution_transactions_by_project_monthly_top5.sql

Tags: production execution transactions

Columns

NameTypeDescription
active_stateUInt64Bitmap representing active accounts associated with the project during the month.
dateDateTimeThe month start date for the aggregated data (UTC).
fee_nativeUInt64Sum of native fees paid for transactions in the month, rounded to six decimal places.
gas_usedUInt64Total gas units used for transactions in the month.
labelStringProject label, with 'Others' for projects outside top 5.
metricStringMetric name identifier (e.g., 'Transactions', 'FeesNative', 'GasUsed', 'ActiveAccounts').
monthDateTime64The first day of the month representing the aggregation period.
projectStringIdentifier for the project associated with the transactions; 'Others' groups projects outside the top 5.
txsUInt64Total number of transactions for the project in the month.
valueFloat64Corresponding metric value for the project and month.

Dimensions

  • month (time)
  • project (categorical)
  • date (time)
  • label (categorical)
  • metric (categorical)

Measures

  • fee_native_value (sum)

  • active_state_value (sum)
  • value_value (sum)
