int_mixpanel_ga_events_daily

Daily event counts by event_name and event_category for Gnosis App (app.gnosis.io) production traffic only. Tracks total events, unique users, unique devices, and autocapture ratio per event type per day. event_category groups events into: pageview (automatic $mp_web_page_view), modal (Open Modal / Open Bottom Sheet), login (Login with Passkey), feature (Success - Circles mint, Marketplace Purchase, QR Scanner events), navigation (back/Back/Close ariaLabel clicks), system ($ prefix events), action (other autocaptured interactions), other. Deploy previews, localhost, and kitchen.app.gnosis.io are excluded.
