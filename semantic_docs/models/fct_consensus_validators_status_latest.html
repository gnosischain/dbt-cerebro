fct_consensus_validators_status_latest

Validator-level point-in-time snapshot at the most recent slot. One row per validator with current balance, effective balance, status, slashed flag, lifecycle epochs (activation/eligibility/exit/withdrawable), and a derived 20-byte withdrawal_address parsed from 0x01/0x02 credentials (NULL for 0x00 BLS credentials). Sourced from stg_consensus__validators_all by selecting MAX(slot). Materialized as a table; the upstream API view (api_consensus_validators_status_latest) is the canonical lookup endpoint by pubkey / validator_index / withdrawal_credentials / withdrawal_address.
