int_execution_transactions_info_daily

The `int_execution_transactions_info_daily` model aggregates daily transaction data from the execution layer, providing insights into transaction counts, success rates, gas consumption, and value metrics for analysis and reporting.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/transactions/intermediate/int_execution_transactions_info_daily.sql

Tags: production execution transactions

Columns

NameTypeDescription
block_timestampDateTime64The exact timestamp when the transaction was recorded on the blockchain.
dateDateThe date (YYYY-MM-DD) extracted from `block_timestamp`, used for daily aggregation.
fee_native_sumFloat64Total fee paid in native units (XDAI).
fee_usd_sumFloat64Total fee in USD, calculated using price data.
gas_priceUInt64The gas price paid per gas unit in wei, represented as a float.
gas_price_avgFloat64Average gas price in Gwei.
gas_price_medianFloat64Median gas price in Gwei.
gas_usedUInt64The total amount of gas units consumed by the transaction, as a float.
n_txsUInt64Total number of transactions of this type on the day.
successUInt8Indicator (0 or 1) denoting whether the transaction was successful.
transaction_typeStringThe category or type of transaction, represented as a string.
value_nativeFloat64The transaction value in native token units (xDai), expressed as a float with decimal precision.
xdai_valueFloat64Sum of transaction values in native units (XDAI).
xdai_value_avgFloat64Average transaction value in native units.
xdai_value_medianFloat64Median transaction value in native units.
