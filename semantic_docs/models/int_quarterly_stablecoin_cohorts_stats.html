int_quarterly_stablecoin_cohorts_stats

Pre-computed quarterly statistics (min, max, avg, median) for stablecoin holder cohorts by balance bucket and peg class. Aggregates daily cohort data across all days in each quarter. Excludes BRZ. Materialized as a table for fast API reads.
