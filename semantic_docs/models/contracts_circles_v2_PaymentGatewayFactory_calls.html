contracts_circles_v2_PaymentGatewayFactory_calls

Decoded function calls for the Circles v2 PaymentGatewayFactory contract. This model is decoded from `execution.traces` (not `execution.transactions`) because createGateway is invoked through a Safe in the ERC-4337 bundler path, so the Factory selector is only visible one hop down in internal calls. decode_calls auto- detects the traces source and emits an extra `trace_address` column used in the unique_key.
