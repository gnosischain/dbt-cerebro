stg_pools__balancer_v3_pool_tokens

Ordered token list per Balancer V3 pool derived from Swap events. Balancer V3 stores tokens in ascending address order; ROW_NUMBER() over sorted unique token addresses reproduces the same 0-based index used in LiquidityAdded/LiquidityRemoved amountsRaw arrays. Tokens are the ERC4626 wrappers the pool actually holds (e.g. waGnoGNO, waGnoWETH) — resolve to underlying via stg_pools__balancer_v3_token_map for price lookups. Only pools with at least one Swap event appear; pools with deposited liquidity but zero swaps are excluded (negligible activity in practice).
