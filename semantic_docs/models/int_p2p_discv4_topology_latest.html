int_p2p_discv4_topology_latest

The model captures the latest peer-to-peer discovery topology by aggregating neighbor relationships and associated metadata, facilitating network analysis and monitoring of discv4 peers.

Module: p2p

Status: candidate

Owner: analytics_team
