api_esg_estimated_nodes_daily

The api_esg_estimated_nodes_daily model provides daily estimates of observed and projected node counts related to ESG carbon footprint analysis, facilitating trend analysis and uncertainty quantification.

Module: ESG

Status: candidate

Owner: analytics_team

Path: models/ESG/marts/api_esg_estimated_nodes_daily.sql

Tags: production esg tier1 api:estimated_nodes granularity:daily

Columns

NameTypeDescription
baseline_observed_nodesFloat64The observed number of nodes based on baseline measurements on the given date.
dateDateThe specific calendar date for the recorded data point.
estimated_nodesUInt64The model's estimated total number of nodes for the given date.
nodes_lower_95UInt64The lower bound of the 95% confidence interval for the estimated node count.
nodes_upper_95UInt64The upper bound of the 95% confidence interval for the estimated node count.
