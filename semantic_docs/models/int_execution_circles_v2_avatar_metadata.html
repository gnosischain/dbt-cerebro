int_execution_circles_v2_avatar_metadata

Current parsed IPFS metadata per Circles v2 avatar. Joins each avatar's most recent metadata digest to the latest fetched payload in `circles_avatar_metadata_raw`.

Module: execution

Status: docs_only

Owner: analytics_team

Path: models/execution/Circles/intermediate/int_execution_circles_v2_avatar_metadata.sql

Tags: production execution circles_v2 avatar_metadata

Columns

NameTypeDescription
avatarStringLowercase Circles avatar address.
avatar_typeStringAvatar category (Human, Group, Org).
gateway_urlNullable(String)Gateway URL fetched at materialization time.
ipfs_cid_v0Nullable(String)CIDv0 of the current metadata payload.
metadata_bodyNullable(String)Full raw response body for the current metadata digest.
metadata_descriptionString`description` field extracted from the metadata JSON. Populated for ~20% of payloads.
metadata_digestNullable(String)0x-prefixed hex digest of the current metadata payload.
metadata_fetched_atNullable(DateTime)Timestamp the current payload was fetched.
metadata_image_urlString`imageUrl` field from the metadata JSON. Populated for ~1.5% of payloads.
metadata_nameString`name` field extracted from the metadata JSON. Populated for ~99.9% of fetched payloads.
metadata_preview_image_urlString`previewImageUrl` field from the metadata JSON. Populated for ~53% of payloads.
metadata_symbolString`symbol` field extracted from the metadata JSON. Rarely populated (group/org tokens only).
onchain_nameNullable(String)Name set on-chain at registration (Group/Org only).
