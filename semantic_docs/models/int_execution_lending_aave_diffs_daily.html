int_execution_lending_aave_diffs_daily

Daily per-user scaled balance deltas for Gnosis lending markets (Aave V3 and SparkLend). Computes scaled deltas from Pool events (Supply, Withdraw, Repay with aTokens, LiquidationCall where receiveAToken=false) and aToken BalanceTransfer events, aggregated per (date, protocol, user, reserve). Scaled math uses exact UInt256 WadRayMath and each pool action is paired with the ReserveDataUpdated snapshot at the same log_index position within its (protocol, transaction). LiquidationCall rows where receiveAToken=true are intentionally excluded because the collateral movement is already captured by the BalanceTransfer(borrower -> liquidator) event.
