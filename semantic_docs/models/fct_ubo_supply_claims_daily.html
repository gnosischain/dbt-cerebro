fct_ubo_supply_claims_daily

THE reusable UBO supply-claims surface. One row per (date, protocol, container_address, ubo_address) — for each container contract that pools a token, the individual end-holders and their withdrawable balance. A "supply claim" is a withdrawable position an end-holder has on a token held by a container contract (aToken, vault, pool LP). Protocols: Aave V3 + SparkLend, Balancer V2, Uniswap V3, Swapr V3, Curve 3pool, sDAI. New UNION ALL branches can be added; downstream consumers pick them up automatically. Use this model to answer "who really holds this token" questions — join with int_execution_tokens_balances_daily on (token_address, address) to replace container-contract rows with their UBOs.
