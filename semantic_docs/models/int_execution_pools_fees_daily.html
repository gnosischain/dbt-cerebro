int_execution_pools_fees_daily

Daily accrued pool fees and trading volume at pool×token grain for Uniswap V3 and Swapr V3 pools, computed from Swap + Flash events (gross fees; independent of Collect/claims). V3 swap/flash deltas are sourced from staging models (stg_pools__uniswap_v3_events, stg_pools__swapr_v3_events) which handle int256 two's-complement decoding. Fees are applied in ppm (1e6 denominator). Volume = gross incoming token amount per swap; flash loans contribute 0 volume. Swapr V3 (Algebra) fees are dynamic via Fee events — the latest fee is applied as-of each swap, with swaps before the first Fee event backfilled using the pool's first observed fee.
