int_esg_node_classification

The `int_esg_node_classification` model aggregates and classifies peer nodes based on their observed attributes and operational archetypes, supporting ESG and network analysis initiatives.

Module: ESG

Status: candidate

Owner: analytics_team

Path: models/ESG/intermediate/int_esg_node_classification.sql

Tags: production esg nodes

Columns

NameTypeDescription
avg_confidenceFloat64Average classification confidence score for the category
calculated_atDateTime64(0, 'UTC')Timestamp when the data was calculated
category_percentageFloat64Percentage share of the category within total estimated nodes per day
classification_confidenceFloat64Confidence score of the classification (0 to 1)
client_typeStringType of client used by the peer node, as identified during the last visit.
country_codeStringISO code representing the country where the peer node was located at last observation.
dateDateTime64(0, 'UTC')Date corresponding to the distribution data
estimated_total_nodesUInt64Estimated total number of nodes in the category after scaling
generic_providerStringThe cloud or network provider associated with the peer node, indicating hosting or transit services.
geographic_distributionStringJSON string of country codes and their estimated node counts
hidden_nodes_estimatedUInt64Estimated number of hidden nodes not directly observed
hidden_nodes_percentageFloat64Percentage of hidden nodes relative to total estimated nodes
ip_addressStringThe IP address associated with the peer node during the last observation.
last_seen_that_dayDateTime64(0, 'UTC')Timestamp of the most recent visit to the peer node on the observation date.
node_categoryStringClassified category of the node, e.g., 'cloud_hosted', 'professional_operator'
nodes_lower_95UInt64Lower bound estimate of total nodes at 95% confidence
nodes_upper_95UInt64Upper bound estimate of total nodes at 95% confidence
observation_dateDateThe date when the peer node was last observed, formatted as a date.
observed_nodesUInt64Count of observed nodes for the category on the date
peer_idUInt64Unique identifier for each peer node observed in the network.
peer_orgStringOrganization name associated with the peer node, used for classification purposes.
sample_coverageFloat64Estimated sample coverage (proportion of total nodes observed)
scaling_factorFloat64Scaling factor applied to observed nodes based on reachability data
top_countriesStringComma-separated list of top 5 countries by estimated node count
