api_execution_circles_v2_avatar_token_distribution

For each Circles v2 avatar's personal CRC token (the ERC-1155 token whose token_address equals the avatar address), classifies all current holders into Self / Wrapped (ERC-20) / Other Circles avatars / Other contracts. One row per (avatar, holder_category) — the dashboard's avatar global filter selects which avatar's token distribution to inspect.
