fct_execution_gnosis_app_gpay_migration_daily

Daily Gnosis Pay exploit -> Safe migration recovery time-series (June-2026 incident). One row per calendar date from the 2026-06-04 cutover through the latest observed data, backing the Migration tab KPIs / line charts. The 66,001 OLD->NEW safe pairs come 1:1 from int_execution_gpay_safe_canonical; refund USD is priced with the same per-token USD/native ratio as int_execution_gpay_balances_daily, so refund USD reconciles with the balance USD series.
