int_execution_gnosis_app_user_events_unified

Thin UNION ALL of `int_execution_gnosis_app_events_chain_unified` + `int_execution_gnosis_app_events_mixpanel_unified`. The single touchpoint table the MTA persona's runtime mapping points at as `touchpoint_model`. Mixpanel rows are pseudonymized at ingestion using the same salt as the bridge — equality on `user_pseudonym` joins chain-side actions to in-app sessions for the same human without ever materializing a raw address. Chain rows have NULL for Mixpanel-only columns (`device_type`, `country_code`, `page_path`, `bottom_sheet`); Mixpanel rows have NULL for `amount_usd`. Otherwise the schemas match column-for-column. Incremental, partitioned by month on `event_date`. Low cost — the upstreams already enforce incremental boundaries.
