int_consensus_validators_balances_daily

The `int_consensus_validators_balances_daily` model aggregates daily total and effective balances of validators from the consensus layer, supporting historical balance analysis and reporting.

Module: consensus

Status: candidate

Owner: analytics_team

Path: models/consensus/intermediate/int_consensus_validators_balances_daily.sql

Columns

NameTypeDescription
balanceUInt64The total sum of validator balances in ETH, expressed as a decimal, representing the combined balance at the start of each day.
dateDateTime64The start of each day (UTC) for which validator balances are aggregated.
effective_balanceUInt64The sum of validator effective balances in ETH, expressed as a decimal, indicating the active stake at the start of each day.
