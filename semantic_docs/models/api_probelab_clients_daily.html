api_probelab_clients_daily

The api_probelab_clients_daily model aggregates daily maximum crawl creation dates and agent version types for Probelab clients, providing insights into client activity and agent version distribution over time.

Module: probelab

Status: candidate

Owner: analytics_team

Path: models/probelab/marts/api_probelab_clients_daily.sql

Tags: production probelab clients tier1

Columns

NameTypeDescription
clientStringThe type of agent version used by the client, indicating the client category or version type.
dateDateTime64The start of the day for which the data is aggregated, representing the date of maximum crawl creation.
valueInt32The integer count of total crawls, floored to the nearest whole number, representing activity volume per client per day.

Dimensions

  • date (time)
  • client (categorical)

Measures
