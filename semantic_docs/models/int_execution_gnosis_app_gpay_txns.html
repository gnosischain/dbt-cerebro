int_execution_gnosis_app_gpay_txns

Any USER-INITIATED Gnosis Pay card-wallet transaction (Payment = card spend, Crypto Withdrawal, Fiat Off-ramp, Fiat Top Up) by a currently GA-owned wallet, from the GA launch (2025-11-12) on, attributed to the safe's first GA owner (ga_user) via the int_execution_gnosis_app_gpay_wallets bridge. Structural twin of int_execution_gnosis_app_gpay_topups; excludes 'Crypto Deposit' (already the topup leg, so the two gPay legs stay disjoint) and system-side 'Reversal' / 'Cashback' (not user activity).
