int_execution_circles_v2_crc20_pools

View of all Uniswap V3 and Swapr V3 pools that contain at least one CRC20 wrapper token. Combines the dynamic pool registry (all pools ever created by the factories) with the ERC20Lift wrapper list, so it captures CRC20 pools without requiring them to be in contracts_whitelist.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/Circles/intermediate/int_execution_circles_v2_crc20_pools.sql

Tags: production execution circles_v2 prices

Columns

NameTypeDescription
avatarStringCircles avatar address that owns/minted this CRC20 wrapper.
backing_tokenStringAddress of the other token in the pool (the pricing token, e.g. WXDAI, WETH).
circles_typeUInt8Wrapper type — 0 = demurraged (decaying balance), 1 = inflationary (static balance, typically used in AMMs).
crc20_is_token0BoolTrue if the CRC20 wrapper is token0 in the pool, false if it is token1.
crc20_symbolNullable(String)Display symbol for the CRC20 token, derived from dune_labels or a fallback abbreviation.
crc20_tokenStringAddress of the CRC20 wrapper token in this pool.
pool_addressStringPool contract address (lowercase, 0x-prefixed).
pool_address_no0xStringPool contract address without 0x prefix, for direct comparison with execution.logs.address.
protocolStringDEX protocol name (Uniswap V3 or Swapr V3).
