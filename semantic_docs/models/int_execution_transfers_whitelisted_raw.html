int_execution_transfers_whitelisted_raw

This dbt model processes and enriches whitelisted ERC20 token transfer logs, integrating token metadata and price data to facilitate detailed transfer analysis within a specified date range.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/transfers/intermediate/int_execution_transfers_whitelisted_raw.sql

Tags: dev execution transfers erc20 whitelisted

NameTypeDescription
