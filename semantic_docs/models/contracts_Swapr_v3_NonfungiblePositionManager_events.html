contracts_Swapr_v3_NonfungiblePositionManager_events

Decoded events from the Swapr V3 (Algebra) NonfungiblePositionManager contract (0x91fd594c46d8b01e62dbdebed2401dde01817834) on Gnosis Chain. Decodes IncreaseLiquidity, DecreaseLiquidity, Transfer (ERC-721), and Collect events. Used by the UBO pipeline to attribute pool liquidity positions to individual LPs via tokenId ownership tracking. Swapr's IncreaseLiquidity event includes the pool address directly, simplifying the tokenId-to-pool mapping vs Uniswap V3.
