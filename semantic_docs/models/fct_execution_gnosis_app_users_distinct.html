fct_execution_gnosis_app_users_distinct

One row per Gnosis App on-chain user_pseudonym, with boolean flags indicating which identification heuristic(s) fired and a high- confidence flag (n_distinct_heuristics >= 2). user_pseudonym is the canonical cross-domain join key (sipHash64 of lowercased address with the project PII salt). Same hash space as: fct_revenue_per_user_*, fct_execution_gpay_users_distinct, and stg_mixpanel_ga__events.user_id_hash. Powers cross-sector overlap queries through the semantic layer. "Gnosis App" spans two complementary data domains: web/mobile analytics (Mixpanel-sourced, models/mixpanel_ga/) and on-chain behaviour (this module). Discovery queries for "Gnosis App users" should surface metrics from both — synonyms on the semantic_model are set up accordingly.
