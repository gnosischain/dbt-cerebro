stg_pools__balancer_v3_token_map

Balancer V3 ERC4626 wrapper → underlying token mapping. Used to resolve wrapped Aave tokens (waGno*) to their underlying for price lookups and token symbol resolution. Hardcoded for the 5 Aave wrappers on Gnosis Chain; wrapper tokens are ~1:1 with underlying (plus accrued yield).
