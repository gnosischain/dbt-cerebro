api_esg_info_category_daily

The api_esg_info_category_daily model aggregates daily ESG-related metrics, including carbon footprint, energy consumption, and node counts, categorized by stakeholder type for analytical insights.

Module: ESG

Status: candidate

dateDateThe date for which the ESG metrics are recorded, in date format.
labelStringThe stakeholder group associated with the metrics, such as Home Staker, Professional Validator, Cloud Provider, or Unknown.
