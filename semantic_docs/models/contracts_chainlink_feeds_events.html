contracts_chainlink_feeds_events

Decoded Chainlink price-feed AnswerUpdated events for all aggregators used on Gnosis (GNO/USD, ETH/USD, WBTC/USD, EUR/USD, CHF/USD, wstETH-ETH, USDC/USD, USDT/USD, DAI/USD; both phase aggregators per feed). Single-pass decode of the address set; contract_address identifies the feed.
