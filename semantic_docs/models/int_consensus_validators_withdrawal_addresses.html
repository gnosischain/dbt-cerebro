int_consensus_validators_withdrawal_addresses

Thin view over int_consensus_validators_labels that derives the controlling EVM address from each validator's withdrawal_credentials (trailing 20 bytes for 0x01-type credentials). Saves downstream queries the substring extraction and enables direct joins from consensus validators to execution-side address-bearing tables.
