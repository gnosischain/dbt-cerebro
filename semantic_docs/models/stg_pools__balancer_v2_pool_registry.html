stg_pools__balancer_v2_pool_registry

Mapping of Balancer V2 poolId (bytes32) → pool contract address, built from PoolRegistered events on the Vault. Materialized as a small table so that downstream views — especially the live-trades staging model queried on every dashboard poll — can join against it cheaply instead of re-scanning the full historical Vault events table.
