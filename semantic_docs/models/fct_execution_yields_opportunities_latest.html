fct_execution_yields_opportunities_latest

Combined yield opportunities table for LP pools and lending markets. Ranked by yield for quick comparison.

Module: execution

Status: candidate

Owner:

Path: models/execution/yields/marts/fct_execution_yields_opportunities_latest.sql

Tags: production execution yields opportunities latest

Columns

NameTypeDescription
addressStringContract address. For LPs the pool contract; for lending the reserve token contract. Used for Blockscout links.
borrow_apyFloat64Variable borrow APY (lending only, NULL for LPs).
fee_pctNullable(Float64)Pool swap fee as a percentage (e.g. 0.3 = 0.3%). Static for Uniswap V3, latest dynamic fee for Swapr V3 and Balancer V3. NULL for lending markets.
fees_7dFloat64Fees accrued over the last 7 days in USD (LP only, NULL for lending).
nameStringDisplay name. For LPs the pair and address suffix; for lending the token symbol.
net_apr_7dFloat64Net APR after LVR (fee_apr_7d + lvr_apr_7d). LP only, NULL for lending.
pool_keyNullable(String)Exact full pool label for LP opportunities, matching the labels used by pool-filtered metrics. NULL for lending markets.
protocolStringProtocol name (e.g. Uniswap V3, Swapr V3, Aave V3).
rate_trend_14dArray(Float64)Last 14 daily yield points ordered oldest to newest. LP rows use fee_apr_7d; lending rows use supply APY.
tokenStringBase token symbol (e.g. WETH, USDC.e).
total_borrowedFloat64Total borrowed in USD (lending only, NULL for LPs). Derived from total_supplied * utilization_rate / 100.
total_suppliedFloat64Total supplied in USD (lending only, NULL for LPs). Sum of all deposit positions.
tvlFloat64Total value locked in USD (LP pools only, NULL for lending).
typeStringOpportunity type: LP or Lending.
utilization_rateFloat64Lending utilization rate as a percentage (total_borrows / total_supply * 100). Lending only, NULL for LPs. High values (>80%) indicate withdrawal risk.
volume_usd_7dFloat64Total trading volume in USD over the last 7 days (LP only, NULL for lending).
yield_labelStringYield type label: APR (LP) or APY (Lending).
yield_pctFloat64Current yield percentage (fee APR for LPs, supply APY for lending).
