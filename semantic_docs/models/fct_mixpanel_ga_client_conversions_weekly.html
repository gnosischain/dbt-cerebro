fct_mixpanel_ga_client_conversions_weekly

Weekly progression of client-side (Mixpanel) conversions by UTM — card_order_started / card_ordered / crc_minted / circles_created etc., seed-driven via seeds/mixpanel_conversion_events.csv (event-name steps) and seeds/mixpanel_conversion_pages.csv (page-reach proxies: card_ordered = /gnosis-pay/kyc; the order flow emits no completion event). Complements the on-chain steps so Growth's full funnel by UTM lives in one place. Tidy/long shape: attribution_model = first_touch | last_touch (stacked long — filter to one value before summing new_users, or counts double). Aggregate-only.
