api_execution_trades_stats_size_distribution

Dashboard view — trade-size distribution per time window. Percentages are computed within each window (1m/3m/6m/1y) off the pre-bucketed `size_bucket` column in `int_execution_trades_by_tx`. The dashboard filters by time_window.
