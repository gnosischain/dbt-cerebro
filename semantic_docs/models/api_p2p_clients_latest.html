api_p2p_clients_latest

This view provides the latest counts and percentage changes of P2P clients for discv4 and discv5 protocols, enabling monitoring of client growth trends over the most recent week.

Module: p2p

Status: candidate

Owner: analytics_team

Path: models/p2p/marts/api_p2p_clients_latest.sql

Tags: production p2p tier0 api:clients_count granularity:latest

Columns

NameTypeDescription
change_discv4_pctFloat64Percentage change in Discv4 clients compared to 7 days prior.
change_discv5_pctFloat64Percentage change in Discv5 clients compared to 7 days prior.
dateDateThe date corresponding to the latest available data for the client counts.
discv4_countUInt64The total number of discv4 clients recorded on the latest date.
discv5_countUInt64Number of Discv5 clients for the latest date.
