fct_execution_transactions_by_project_snapshots

This model aggregates transaction, fee, and active account data by project over various time windows, enabling comparative analysis of recent and historical performance metrics.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/transactions/marts/fct_execution_transactions_by_project_snapshots.sql

Tags: production execution transactions

Columns

NameTypeDescription
bucketStringThe project or bucket identifier.
change_pctFloat64Percentage change from previous window, null if not applicable.
labelStringThe label indicating the metric type, e.g., 'Transactions', 'FeesNative', 'ActiveAccounts'.
max_dateDateThe most recent date available in the daily transaction data, used as the reference point for calculating rolling and historical windows.
valueFloat64The metric value, converted to float64 for numerical analysis.
windowStringThe time window identifier, e.g., '1D', '7D', '30D', '90D', or 'All'.

Dimensions

  • max_date (time)
  • label (categorical)
  • window (categorical)
  • bucket (categorical)

Measures

  • value_value (sum)
  • change_pct_value (average)
