int_execution_zodiac_modifier_module_events

Long-form append log of every EnabledModule / DisabledModule event emitted BY a Zodiac Modifier proxy (Delay / Roles) — i.e. the sub-modules enabled or disabled ON the Modifier itself. The Zodiac Modifier emits EnabledModule(address module) NON-indexed (module in data), unlike Gnosis Safe which emits it indexed. Decoded via decode_logs against contracts_zodiac_modules_registry, which resolves the mastercopy ABI (the non-indexed EnabledModule/DisabledModule rows were added to the event_signatures seed for every vulnerable mastercopy). This is NOT the same as int_execution_safes_module_events, which is SafeProxy-scoped and therefore never sees Modifier-emitted events. decode_logs handles its own incremental filter; do NOT add an outer block_timestamp filter.
