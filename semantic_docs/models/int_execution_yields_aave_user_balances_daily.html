int_execution_yields_aave_user_balances_daily

Daily per-user aToken balances for Aave V3 on Gnosis. Reads pre-computed scaled deltas from int_execution_yields_aave_diffs_daily, builds a calendar of all active user-reserve pairs, computes running cumulative scaled balances, then converts to underlying amounts using end-of-day liquidityIndex. Scoped to reserves in atoken_reserve_mapping seed.
