int_quarterly_esg_carbon_intensity_with_fallback

Forward-fills Ember carbon intensity data for periods where values are missing (Jan 2026+). Carries the last known carbon intensity per country for up to 12 months. UNIONs real data from int_esg_carbon_intensity_ensemble with estimated rows.

Module: quarterly_data

Status: approved

Owner: analytics_team

Path: models/quarterly_data/esg/intermediate/int_quarterly_esg_carbon_intensity_with_fallback.sql

Tags: production quarterly_data esg carbon_intensity

Columns

NameTypeDescription
carbon_intensity_meanFloat64Point estimate of carbon intensity in gCO2/kWh, seasonally adjusted.
carbon_intensity_stdFloat64Standard deviation of carbon intensity estimate.
country_codeStringISO 3-letter country code or 'WORLD' for global average.
is_estimatedBoolTrue if the value is forward-filled from the last known Ember data rather than a real measurement.
month_dateDateMonth start date for the carbon intensity measurement or estimate.

Dimensions

  • country_code (categorical)
