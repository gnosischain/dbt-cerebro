api_mixpanel_ga_pages_daily

Not exposed via cerebro-api; accessible to the internal MCP / semantic layer. Daily Gnosis App page-path analytics. One row per (date, current_domain, page_path) with event and unique-user counts. Sourced from int_mixpanel_ga_pages_daily. Use case: per-page traffic analysis and top-pages widgets. Page paths are stripped of query parameters upstream so they are stable across campaigns.
