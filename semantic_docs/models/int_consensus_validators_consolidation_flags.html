int_consensus_validators_consolidation_flags

Small per-validator lookup: does this validator appear as the SOURCE or TARGET of any cross-consolidation request (ever). Self-consolidations are excluded because they don't affect balance attribution. Materialized as a table so downstream per-batch income_daily queries don't have to re-run the consolidation_requests × status_latest join on every monthly batch (which hits the 10.8 GiB cluster memory cap on the shared tier).
