int_execution_mmm_media_weekly

Long-form weekly MMM media (incentive / reward / outlay) registry. One row per (`week`, `media_name`) on the same continuous weekly spine as the KPI registry. Missing weeks filled with 0 for sum-method outlays and NULL for `weighted_avg` / `avg` snapshots. Source aggregators (weekly grain): validator_proposer_rewards_gno ← sum(proposer_reward_total_gno) over int_consensus_validators_proposer_rewards_daily ga_token_offer_emissions_usd ← sum(amount_received_usd) over int_execution_gnosis_app_token_offer_claims pools_lp_fee_apr_avg ← TVL-weighted mean of fee_apr_7d (sum(fee_apr_7d * tvl_usd) / sum(tvl_usd)) over int_execution_pools_metrics_daily Adding a new media: append to seed `mmm_media_registry.csv` AND add an aggregator CTE here AND add a `sumIf` clause in the wide pivot. Incremental on `week` (monthly partition).
