api_quarterly_data_stablecoin_holders

Quarterly stablecoin holder statistics on Gnosis Chain, split by peg class. Shows median, max, and average of daily holder count across all days in the quarter. Note: holders are summed across tokens so an address holding multiple stablecoins is counted once per token. Excludes BRZ.

Module: quarterly_data

Status: approved

Owner: analytics_team

Path: models/quarterly_data/stablecoins/marts/api_quarterly_data_stablecoin_holders.sql
