fct_bridges_sankey_edges_token_daily

The fct_bridges_sankey_edges_token_daily model captures daily token transfer volumes across bridges, illustrating flow directions between chains for Sankey diagram visualizations.

Module: bridges

Status: candidate

Owner: analytics_team

Path: models/bridges/marts/fct_bridges_sankey_edges_token_daily.sql

Tags: production bridges fct

Columns

NameTypeDescription
bridgeStringThe name or identifier of the bridge facilitating the token transfer.
dateDateThe date of the recorded token transfer data, formatted as YYYY-MM-DD.
dest_chainStringThe destination blockchain network for the token transfer.
directionStringThe direction of the token flow, either 'in' or 'out'.
sideStringThe side of the token flow, either 'left' or 'right'.
sourceStringThe originating chain or bridge in the token flow.
source_chainStringThe originating blockchain network for the token transfer.
targetStringThe destination chain or bridge in the token flow.
tokenStringThe specific token being transferred, identified by its symbol or address.
valueFloat64The total volume of the token transferred, measured in USD.
volume_usdUInt64The total transfer volume in USD for the given date, source, destination, bridge, and token.
