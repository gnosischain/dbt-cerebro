int_execution_gnosis_app_bundlers

Dynamic Cometh ERC-4337 bundler allowlist for Gnosis App attribution. Supersedes the hand-maintained seeds/gnosis_app_relayers.csv, which listed only 3 of ~61 real bundlers (so ~10% of GA users were missed; miss-rate hit 16-22% for Mar-2026+ cohorts when the 0x4337... fleet launched). A bundler qualifies if it sends UserOps to the EntryPoint and either (a) matches the Cometh vanity prefix 0x4337..., or (b) relayed a GA-specific, relayer-agnostic anchor: a CRC fee transfer (ERC-1155 Hub or gCRC ERC-20) to the fee receiver, or a GA InvitationModule RegisterHuman. The curated seed is unioned in (belt-and-suspenders). Self-maintains as the fleet rotates. Schema mirrors gnosis_app_relayers so consumers swap only the ref().
