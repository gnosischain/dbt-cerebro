int_esg_node_geographic_distribution

This model provides a detailed geographic distribution of nodes within different categories, including estimated counts and regional metadata, to support ESG and network analysis.

Module: ESG

Status: candidate

Owner: analytics_team

Path: models/ESG/intermediate/int_esg_node_geographic_distribution.sql

Tags: production esg nodes

Columns

NameTypeDescription
calculated_atDateTimeTimestamp when the data was calculated.
category_percentageFloat64Percentage of total nodes within the node category for the country.
country_codeStringISO alpha-2 country code representing the country.
country_code_alpha3StringISO alpha-3 country code.
country_nameStringThe full name of the country.
dateDateThe date for which the geographic distribution data is applicable, formatted as a date.
estimated_nodesUInt64Estimated number of nodes for the country.
estimated_total_nodesUInt64The total number of nodes estimated for the given category and date.
geographic_distributionStringA JSON array detailing the distribution of nodes across countries, with each entry specifying a country code and its estimated node count.
global_percentageFloat64Percentage of total nodes across all countries for the date.
global_rankUInt64Rank of the country globally based on estimated nodes.
node_categoryStringThe classification category of nodes, indicating different types or segments within the network.
nodes_lower_95UInt64The lower bound of the 95% confidence interval for the estimated total nodes.
nodes_upper_95UInt64The upper bound of the 95% confidence interval for the estimated total nodes.
rank_in_categoryUInt64Rank of the country within the node category based on estimated nodes.
regionStringThe region to which the country belongs.
sub_regionStringThe sub-region within the main region.
