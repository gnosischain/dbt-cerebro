fct_execution_pools_il_daily

Pool-level Loss Versus Rebalancing (LVR) from actual swap flows. LVR measures the cost LPs pay to arbitrageurs — the value extracted via swap rebalancing. Sign convention: negative = loss (arbitrage extracted value), positive = gain. Swap events already embed the concentrated-liquidity math executed by the pool contract, so this captures V3 amplification without per-position tracking. Each day's swap flows are valued at that day's token prices, then summed over the 7-day window. lvr_apr_7d = (swap_flow_usd_7d − fees_usd_7d) / avg_tvl_7d × (365/7) × 100. Fees are subtracted because swap event amounts include fees on the input side.
