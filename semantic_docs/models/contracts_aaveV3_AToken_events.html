contracts_aaveV3_AToken_events

Decoded event logs from Aave V3 aToken contracts on Gnosis (aGnoGNO, aGnoWXDAI, aGnosDAI, aGnoUSDC, aGnoEURe, aGnoUSDCe). Key events include BalanceTransfer (scaled balance movements between users), Transfer (ERC20), Mint, Burn, and Initialized.

Module: contracts

Status: docs_only

Owner: analytics-team

block_numberUInt64Number of the block containing the event.

decoded_paramsMap(String, String)Map of decoded event parameters (keys and values vary by event type).
event_nameStringName of the event emitted, e.g. BalanceTransfer, Transfer, Mint, Burn.
log_indexUInt64Index of the log within the block, used to order events.
