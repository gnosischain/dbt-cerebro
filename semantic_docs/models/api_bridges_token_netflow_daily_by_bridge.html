api_bridges_token_netflow_daily_by_bridge

This view aggregates daily net flow values for each token across individual bridges and provides a combined total for all bridges, supporting analysis of token movement at a daily granularity.

Module: bridges

Status: candidate

Owner: analytics_team

Path: models/bridges/marts/api_bridges_token_netflow_daily_by_bridge.sql

Tags: production bridges tier1 api:netflow_per_token_per_bridge granularity:daily

Columns

NameTypeDescription
bridgeStringThe name or identifier of the bridge; 'All' indicates the aggregate across all bridges.
bridge_orderUInt64An order indicator used to prioritize specific bridge data over the 'All' category in sorting operations.
dateDateThe specific calendar date for the recorded net flow data.
tokenStringThe identifier of the token involved in the net flow calculation.
valueUInt64The net flow value for the token on the given date, representing the total transfer amount.
