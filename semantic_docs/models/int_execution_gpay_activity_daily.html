int_execution_gpay_activity_daily

Incremental model that aggregates Gnosis Pay wallet activity at the daily level, grouped by wallet, action, direction, and token symbol.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/gpay/intermediate/int_execution_gpay_activity_daily.sql

Tags: production execution gpay activity_daily

Columns

NameTypeDescription
actionStringThe type of activity performed (e.g., payment, deposit, withdrawal, cashback).
activity_countUInt64The number of individual transactions in this aggregation.
amountFloat64The decimal-adjusted token amount of the activity.
amount_rawFloat64The raw token amount of the activity before decimal adjustment.
amount_usdFloat64The USD-equivalent value of the activity.
dateDateThe calendar date of the aggregated activity.
directionStringThe direction of the transaction (e.g., inbound, outbound).
symbolStringThe token symbol involved in the activity.
wallet_addressStringThe blockchain address of the Gnosis Pay wallet.
