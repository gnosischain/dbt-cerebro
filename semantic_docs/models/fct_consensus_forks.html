fct_consensus_forks

The fct_consensus_forks model provides a consolidated view of consensus fork details, including their identifiers, versions, and activation times, to support analysis of network upgrades and protocol changes.

Module: consensus

Status: candidate

Owner: analytics_team

Path: models/consensus/marts/fct_consensus_forks.sql

Tags: production consensus forks

Columns

NameTypeDescription
cl_fork_nameStringThe human-readable name of the fork, such as 'Phase0', 'Altair', or 'Bellatrix'.
fork_digestStringA unique identifier representing the specific fork event, derived from its digest value.
fork_epochStringEpoch number at which the fork occurred.
fork_nameStringName of the fork (e.g., Phase0, Altair, Bellatrix, Capella, Deneb, Electra, Fulu).
fork_timeDateTime64(0, 'UTC')Estimated timestamp when the fork occurred, derived from genesis time and epoch.
fork_versionStringVersion string associated with the fork (e.g., 'PHASE0', 'Altair').
