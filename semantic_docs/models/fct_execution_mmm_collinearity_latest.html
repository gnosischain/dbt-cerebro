fct_execution_mmm_collinearity_latest

Pairwise Pearson correlation matrix between MMM media columns computed over the last 730 days of `fct_execution_mmm_spine_weekly`. Refreshed daily. The MMM analyst reads `is_high_collinearity = 1` (`abs(pearson_corr) > 0.9`) directly to identify pairs that need merge / drop / segment per the multicollinearity check in `mmm_analyst`'s SOP. Cross-product is bounded by the media registry length (~10 media today), so the output is at most ~100 rows. Trivial cost. Only unique pairs are emitted (`a.col_name < b.col_name`).
