int_mixpanel_ga_users_daily

Per-user daily activity summary for Gnosis App (app.gnosis.io) production traffic. One row per (date, user_id_hash). Enables exact DAU computation, new-user detection, and wallet identification rate tracking. is_identified=1 means the user connected a wallet address in that session (distinct_id does NOT start with '$device:'). Anonymous users browse via injected device IDs.
