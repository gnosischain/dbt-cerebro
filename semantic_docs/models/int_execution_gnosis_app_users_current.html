int_execution_gnosis_app_users_current

Snapshot of the Gnosis App sector. One row per address that triggered any heuristic, with derived confidence proxy `n_distinct_heuristics`. An address that triggered 3+ independent rules is a high-confidence Gnosis App user. Cheap to fully rebuild every run; no incremental needed.
