int_bridges_flows_daily

The int_bridges_flows_daily model aggregates daily bridge transfer data, providing insights into token volumes, transaction counts, and net USD flows across different blockchain bridges and chains.

Module: bridges

Status: candidate

Owner: analytics_team

Path: models/bridges/intermediate/int_bridges_flows_daily.sql

Tags: production intermediate bridges

Columns

NameTypeDescription
bridgeStringThe name or identifier of the blockchain bridge facilitating the transfer.
dateDateThe date for which the bridge flow data is aggregated, in YYYY-MM-DD format.
dest_chainStringThe destination blockchain network of the token transfer.
directionStringThe transfer direction, indicating whether the flow is inbound or outbound.
net_usdUInt64Net USD flow, representing the difference between inbound and outbound transfers on the given date.
source_chainStringThe originating blockchain network of the token transfer.
tokenStringThe symbol or identifier of the token being transferred.
txsUInt64Number of transactions processed for the specified bridge flow on the given date.
volume_tokenUInt64Total amount of tokens transferred on the given date, in token units.
volume_usdUInt64Total USD value of tokens transferred on the given date.

Dimensions

  • date (time)
  • bridge (categorical)
  • source_chain (categorical)
  • dest_chain (categorical)
  • token (categorical)
  • direction (categorical)

Measures

  • net_usd_value (sum)
  • txs_value (sum)
