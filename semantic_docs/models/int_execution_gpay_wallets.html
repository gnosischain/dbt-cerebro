int_execution_gpay_wallets

Gnosis Pay Safes derived from on-chain events. A Safe qualifies if it has emitted SafeSetup (so it exists on-chain) AND has ever sent a whitelisted ERC-20 transfer to the Gnosis Pay spender 0x4822521E6135CD2599199c83Ea35179229A172EE, excluding addresses in the gpay_operational_wallets seed. Replaces the crawler-sourced stg_gpay__wallets model.
