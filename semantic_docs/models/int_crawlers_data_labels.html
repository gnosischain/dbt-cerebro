int_crawlers_data_labels

The int_crawlers_data_labels model consolidates and categorizes crawler data labels for various blockchain projects, enabling efficient analysis of project types and their associated attributes.

Module: crawlers_data

Status: approved

Owner: analytics_team

Path: models/crawlers_data/intermediate/int_crawlers_data_labels.sql

Tags: production crawlers_data labels

Columns

NameTypeDescription
addressStringThe unique blockchain address associated with the data label, stored in lowercase for consistency.
introduced_atDateTime64The timestamp indicating when the label was introduced or recorded, used for temporal analysis.
projectStringThe name of the blockchain project or protocol, used to identify and categorize the data label.
sectorStringSector category assigned based on project name, such as DEX, Lending, Bridges, etc.

  • introduced_at (time)
