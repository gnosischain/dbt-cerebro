contracts_circles_v2_Migration_calls

Decoded function calls for the Circles v2 Migration contract (0xd44b8dcfbadfc78ea64c55b705bfc68199b56376). Dedicated V1→V2 migration entry point — `migrate(_avatars, _amounts)` burns V1 balances and mints equivalent V2 CRC. The ABI exposes no events (mint emissions surface as Hub TransferSingle from 0x0 with operator = this contract; see int_execution_circles_v2_mint_events where mint_kind = 'migration'). Decoded from `execution.traces` (not `execution.transactions`) because almost every migrate() call is invoked through a user Safe / AA bundler, so the Migration selector is only visible one hop down in internal calls. decode_calls auto-detects the traces source and emits an extra `trace_address` column used in the unique_key.
