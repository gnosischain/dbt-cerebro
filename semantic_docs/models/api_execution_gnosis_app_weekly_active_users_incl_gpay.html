api_execution_gnosis_app_weekly_active_users_incl_gpay

Gnosis App Weekly Active Users (WAU), Gnosis-Pay-inclusive variant of api_execution_gnosis_app_weekly_active_users: 'active' additionally counts any user-initiated Gnosis Pay card-wallet transaction (spend/withdrawal/off-ramp/fiat top-up) attributed to the safe's Gnosis App owner. Latest incomplete week excluded; WEAU remains a strict subset of the base WAU.
