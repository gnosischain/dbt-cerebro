int_execution_pools_dex_trades

DEX swap events enriched with USD prices and transaction context. Source of truth for trade-level analytics with USD values. Reads from int_execution_pools_dex_trades_raw (materialized, month-partitioned) and adds daily USD prices via ASOF joins plus tx_from/tx_to from int_execution_pools_dex_trades_tx_context. The tx context lookup is a separate materialized table (~80 MB per month) instead of multi-GB from the full execution.transactions table, keeping total memory well within limits alongside the ASOF price joins.
