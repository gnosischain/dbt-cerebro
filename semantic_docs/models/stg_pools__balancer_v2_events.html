stg_pools__balancer_v2_events

Normalized signed token deltas for Balancer V2 pools, decoded from Vault events (PoolBalanceChanged, PoolBalanceManaged, Swap, FlashLoan). PoolBalanceChanged rows are unpacked from parallel JSON arrays of tokens and deltas; deltas are int256 but sometimes serialized as their uint256 two's-complement representation, so negative string values are parsed directly while unsigned values are reinterpreted via reinterpretAsInt256(toUInt256OrNull(...)). PoolBalanceManaged rows sum cashDelta + managedDelta with the same two's-complement decoding to give the net token delta. Swap events emit two rows per swap: a positive delta_amount_raw for tokenIn (amountIn) and a negative delta for tokenOut (-amountOut). Output is keyed by pool_id, block_timestamp, transaction_hash, log_index, with token_address and a signed delta_amount_raw (positive = tokens entering pool, negative = leaving), filtered to non-null pool_id, token_address, and delta.
