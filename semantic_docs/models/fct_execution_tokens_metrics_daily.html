fct_execution_tokens_metrics_daily

Daily per-token fact joining supply/holders (from int_execution_tokens_supply_holders_daily), transfer volume/counts/active addresses (from int_execution_tokens_transfers_daily), and USD prices (from int_execution_token_prices_daily). Keyed on (date, token_address). Excludes the current day (today is incomplete). Materialized as a table partitioned by month for fast time-range scans.
