api_execution_gpay_user_top_wallets

API view of the top 50 Gnosis Pay wallets by lifetime payment volume. One row per wallet_address, selected from fct_execution_gpay_user_lifetime_metrics where total_payment_count > 0 and ordered by total_payment_volume_usd then tenure_days descending. Adds as_of_date (max date in int_execution_gpay_activity_daily). Tier1 endpoint (api:gpay_user_top_wallets, granularity:snapshot).
