fct_execution_blocks_gas_usage_monthly

The fct_execution_blocks_gas_usage_monthly model aggregates monthly gas usage and limits for execution blocks to support performance and resource utilization analysis.

Module: execution

Status: candidate

Owner: analytics_team

NameTypeDescription

gas_used_sum_monthlyUInt64Total gas used in execution blocks during the month, measured in units consistent with gas consumption metrics.
monthDateThe first day of the month representing the aggregation period.
