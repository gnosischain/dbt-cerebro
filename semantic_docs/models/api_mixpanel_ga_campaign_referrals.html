api_mixpanel_ga_campaign_referrals

Not exposed via cerebro-api; accessible to the internal MCP / semantic layer. Aggregate-only pass-through view over fct_mixpanel_ga_campaign_referrals. Per first-touch campaign: signups, Circles inviters (earned and full-invite-graph scopes), inviter rates and invites per inviter. Outcome-based (on-chain ACCEPTED invites) — counts run lower than Mixpanel-side invite-link-sharing tiles. Snapshot — clip campaign views to the Mixpanel era (2025-10+).
