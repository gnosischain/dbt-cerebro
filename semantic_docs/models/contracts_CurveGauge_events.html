contracts_CurveGauge_events

Decoded event log stream from the Curve x3CRV gauge deposit contract (0xb721cc32160ab0da2614cc6ab16ed822aeebc101, proxy → implementation 0x9336DA074c4f585a8B59A8C2B77a32b630cde5A1) on Gnosis Chain. Captures Transfer, Deposit, Withdraw and UpdateLiquidityLimit events. Transfer events are used by int_ubo_claims_curve_gauge_daily to track gauge token balances per depositor. Backfilled from 2021-09-01 in 6-month batches.
