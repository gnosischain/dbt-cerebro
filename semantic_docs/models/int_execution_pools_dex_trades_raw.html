int_execution_pools_dex_trades_raw

Raw DEX swap events across all protocols on Gnosis Chain. One row per swap with token metadata (symbol, decimals, human amounts) but no USD prices — price enrichment lives in int_execution_pools_dex_trades. Each protocol's event decoding lives in its own staging model for easy addition of new protocols. Protocols: Uniswap V3 (stg_pools__dex_trades_uniswap_v3), Swapr V3 (stg_pools__dex_trades_swapr_v3), Balancer V2 (stg_pools__dex_trades_balancer_v2), Balancer V3 (stg_pools__dex_trades_balancer_v3). Taker: for Uniswap V3 and Swapr V3, `taker` is the Swap event recipient (correct even when routed through a contract). For Balancer V2/V3, `taker` is NULL because the Swap event does not include a recipient field. When called with start_month/end_month vars each protocol model filters to that month window; otherwise falls back to the incremental filter.
