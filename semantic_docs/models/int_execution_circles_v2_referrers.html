int_execution_circles_v2_referrers

"Users who start referring" — one row per inviter, i.e. each address that appears as invited_by on a Circles v2 Human avatar registration (int_execution_circles_v2_avatars). Carries the first referral timestamp/week, total n_referrals, first_referral_in_app (1 if the first invitee's registration tx was routed through an active Gnosis App bundler via the ERC-4337 EntryPoint), and first_inviter_fee_at — the first time the inviter received a CRC invitation fee per int_execution_circles_v2_inviter_fees (NULL when no fee was observed; fee-less invites still count as referrals).
