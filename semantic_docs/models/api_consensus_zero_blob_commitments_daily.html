api_consensus_zero_blob_commitments_daily

This view provides daily aggregated counts of consensus blocks with and without zero blob commitments, aiding in monitoring blockchain data integrity and activity levels.

Module: consensus

NameTypeDescription

valueInt64The numerical count of blocks, represented as a 64-bit integer, corresponding to the specified label and date.
