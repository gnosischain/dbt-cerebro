fct_execution_mmm_baseline_latest

Per-(KPI, media) baseline KPI: the median KPI value during weeks where the media's adstocked spend sits in the bottom decile. This is the level the KPI sits at when the media is effectively zero — used by `mmm_analyst` as the `baseline_kpi` input that anchors the log-log response curve fit and the contribution-decomposition intercept. Adstock is computed inline as a geometric 8-week running window with λ=0.5 (recent-week weight 1, prior-week 0.5, two-back 0.25, …) per media column. The bottom-decile threshold is `quantile(0.1)(adstock_v)` per (KPI, media) pair. The persona can recompute with a different λ if a fit prefers it; this mart is a canonical starting point. Pairs with fewer than 5 low-spend weeks are dropped (HAVING clause) because the median wouldn't be statistically meaningful — the `n_low_spend_weeks > 5` test guards against schema drift bringing them back.
