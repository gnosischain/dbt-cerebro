int_p2p_discv4_peers

The int_p2p_discv4_peers model aggregates and processes peer visit data from the Discv4 network, enabling analysis of peer behaviors, versions, and connection properties over time.

Module: p2p

Status: candidate

Owner: analytics_team

Path: models/p2p/intermediate/int_p2p_discv4_peers.sql

Tags: production p2p discv4

Columns

NameTypeDescription
agent_versionStringSoftware version string of the peer's agent, often including platform and runtime details.
buildStringBuild identifier extracted from version string.
channelStringChannel or release type derived from version string.
clientStringClient type or platform derived from peer properties.
connect_maddrStringMultiaddr string used for establishing connections with the peer.
crawl_errorStringError message or code if the crawling attempt to the peer failed.
dial_errorsArray(String)Error details encountered during connection attempts to the peer.
generic_providerStringProvider information associated with the peer's IP.
peer_cityStringCity location of the peer, inferred from IP.
peer_countryStringCountry of the peer, inferred from IP.
peer_hostnameStringHostname associated with the peer's IP address.
peer_idUInt64Unique identifier for each peer in the network.
peer_locStringGeographical location details of the peer.
peer_orgStringOrganization associated with the peer's IP address.
peer_propertiesJSONJSON or structured data containing additional properties of the peer, including network identifiers.
platformStringPlatform information parsed from agent version.
runtimeStringRuntime environment information, if available.
variantStringVariant identifier extracted from version string, if available.
versionStringPeer software version, parsed from version string.
visit_ended_atDateTime64Timestamp indicating when the peer visit concluded.
