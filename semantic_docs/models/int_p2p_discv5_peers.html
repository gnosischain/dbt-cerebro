int_p2p_discv5_peers

The `int_p2p_discv5_peers` model consolidates peer visit data to analyze peer properties, fork versions, and network behavior within specified timeframes for the Discv5 protocol.

Module: p2p

Status: candidate

Owner: analytics_team

Path: models/p2p/intermediate/int_p2p_discv5_peers.sql

Tags: production p2p discv5

Columns

NameTypeDescription
agent_versionStringVersion string of the peer's agent software.
buildStringBuild identifier derived from version string.
channelStringChannel information parsed from version string.
cl_fork_nameStringThe human-readable name of the fork associated with the peer, mapped from known fork digests.
cl_next_fork_nameStringHuman-readable name of the next fork version.
clientStringClient identifier parsed from agent version.
crawl_errorLowCardinality(Nullable(String))Error message encountered during crawling, if any.
crawl_idUUIDUnique identifier for the crawl session.
dial_errorsArray(LowCardinality(String))List of dial errors encountered.
fork_digestStringA string identifier representing the specific fork digest of a peer, used to categorize peers by their fork implementation.
generic_providerStringProvider information for the peer.
ipStringExtracted IPv4 address from connect_maddr.
next_fork_versionStringString representation of the next fork version.
peer_cityStringCity location of the peer.
peer_countryStringCountry location of the peer.
peer_hostnameStringHostname of the peer, joined from external IP info.
peer_idStringUnique identifier for the peer.
peer_locStringGeographical location details of the peer.
peer_orgStringOrganization associated with the peer.
platformStringPlatform derived from agent version, normalized to known categories.
runtimeStringRuntime environment parsed from agent version.
variantStringVariant token extracted from agent version.
versionStringVersion token extracted from agent version.
visit_ended_atDateTime64(0, 'UTC')Timestamp when the peer visit ended.
