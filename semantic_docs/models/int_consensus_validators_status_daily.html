int_consensus_validators_status_daily

The int_consensus_validators_status_daily model aggregates daily counts of validators' statuses to monitor network health and validator activity trends over time.

Module: consensus

Status: candidate

Owner: analytics_team

Path: models/consensus/intermediate/int_consensus_validators_status_daily.sql

Tags: production consensus validators_status

Columns

NameTypeDescription
active_validatorsUInt64Number of active validators on the given date.
cntUInt64The number of validators corresponding to each status on the given date.
dateDateTime64The date (at the start of the day) for which validator status counts are recorded.
exited_validatorsUInt64Number of exited validators on the given date.
slashed_validatorsUInt64Number of slashed validators on the given date.
statusStringThe current status of validators, such as 'active', 'pending', or 'exited'.
total_validatorsUInt64Total number of validators considered in the daily snapshot.
