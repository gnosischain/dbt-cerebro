int_execution_prices_oracle_daily

Daily native USD prices derived from Chainlink on-chain oracle feeds (AnswerUpdated events) for whitelisted tokens. One Chainlink feed per token symbol; the latest answer of each day (argMax over block_timestamp) divided by the feed's decimals. This is the native USD anchor intended to replace the externally-ingested Dune price feed; see docs/native_token_prices_build_plan.md.
