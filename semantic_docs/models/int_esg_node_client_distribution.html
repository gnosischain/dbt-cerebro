int_esg_node_client_distribution

This model estimates the distribution of client nodes across different node categories by integrating client counts from multiple sources and applying scaling factors, aiding in understanding network composition and client deployment.

Module: ESG

Status: candidate

Owner: analytics_team

Path: models/ESG/intermediate/int_esg_node_client_distribution.sql

Tags: production esg nodes

Columns

NameTypeDescription
calculated_atDateTimeTimestamp when the metrics were calculated.
category_client_percentageFloat64Percentage of clients within the node category.
client_efficiency_factorFloat64Efficiency factor of the client relative to baseline (1.0).
client_nodes_lower_95UInt64Lower bound estimate (95% confidence interval) for client nodes.
client_nodes_upper_95UInt64Upper bound estimate (95% confidence interval) for client nodes.
client_typeStringType of client, e.g., consensus_Lighthouse, execution_Geth, etc.
dateDateThe date for which the node classification and client distribution data is recorded.
estimated_client_nodesUInt64Estimated number of client nodes in the category.
estimated_total_nodesUInt64The estimated total number of nodes in the category on the given date.
global_client_percentageFloat64Percentage of clients across all categories.
global_rankUInt64Rank of the client across all categories based on estimated client nodes.
node_categoryStringThe classification category of nodes, such as consensus or execution types.
nodes_lower_95UInt64The lower bound (95% confidence interval) estimate of total nodes in the category.
nodes_upper_95UInt64The upper bound (95% confidence interval) estimate of total nodes in the category.
rank_in_categoryUInt64Rank of the client within its node category based on estimated client nodes.
scaling_factorFloat64A factor used to adjust node counts based on external metrics or model calibration.
