contracts_wxdai_events

The `contracts_wxdai_events` model captures and processes event logs related to the specified contract address, enabling analysis of contract interactions on a monthly incremental basis.

Module: contracts

Status: docs_only

Owner: analytics_team

Path: models/contracts/tokens/contracts_wxdai_events.sql

Tags: production contracts wxdai events

Columns

NameTypeDescription
amountStringThe amount involved in the event, in wei.
block_timestampDateTimeThe timestamp of the block containing the event, in UTC.
contract_addressStringThe address of the contract emitting the event.
event_typeStringThe type of wxDai event (e.g., deposit, withdrawal).
log_indexUInt64The index of the log within the block, used to order events.
recipientStringThe address receiving the tokens, if applicable.
senderStringThe address initiating the transaction.
transaction_hashStringThe hash of the transaction that emitted the event.
