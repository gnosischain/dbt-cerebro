int_execution_circles_v2_mint_events

One row per protocol mint, tagged with `mint_kind`. Sourced from the DECODED Hub events in contracts_circles_v2_Hub_events — NOT inferred from from=0x00…00 TransferSingle legs. Classifier: personal - PersonalMint event. to_address / token_address = the minting human; amount_raw = event `amount`. Replaces the old from=0x0 + Human-token heuristic, which over-counted ~4.5k avatars whose CRC was created only by inline auto-issuance legs that emit no PersonalMint event. Now matches Dune query_6317871 / hub_evt_personalmint (~18.4k distinct humans). group - GroupMint event. to_address = receiver, token_address = group, operator = sender, amount_raw = Σ event `amounts[]`. migration - V1→V2 migration mints have no Hub mint event, so this branch stays transfer-derived: from=0x0 TransferSingle whose operator is the Circles V2 Migration contract (contracts_circles_registry.contract_type = 'Migration'). Incremental, monthly partitioned on block_timestamp.
