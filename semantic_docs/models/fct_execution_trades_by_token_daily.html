fct_execution_trades_by_token_daily

Daily per-token DEX activity. Each swap contributes to BOTH its sold and bought side, so `combined_usd` across all tokens sums to ~2x total chain volume. Suited for "token activity" views — shows every time a token was involved in a trade.
