int_p2p_discv5_clients_daily

The int_p2p_discv5_clients_daily model aggregates daily peer discovery data for P2P clients using the DiscV5 protocol, providing insights into client distribution, platform usage, provider sources, and geographic locations.

Module: p2p

Status: candidate

Owner: analytics_team

Path: models/p2p/intermediate/int_p2p_discv5_clients_daily.sql

Tags: production p2p discv5

Columns

NameTypeDescription
clientStringThe name or type of the client software used by the peer, with 'Unknown' assigned if not specified.
dateDateTime64The specific day for which the aggregated metrics are recorded, representing the start of the day.
generic_providerStringThe provider or source through which the peer's client was obtained or identified.
labelStringLabel corresponding to the metric, e.g., client name, platform, provider, or country.
metricStringType of metric: 'Clients', 'Platform', 'Provider', or 'Country'.
peer_countryStringThe country where the peer is located, based on geographic data.
peer_idUInt64Unique identifier for each peer encountered during the discovery process.
platformStringThe platform (e.g., OS or environment) on which the peer's client is running.
valueUInt64Count of peers or occurrences for the given label on the specified date.
