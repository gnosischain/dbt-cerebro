int_execution_accounts_label_contracts

Protocol / token / infra contract addresses identified from Dune labels (int_crawlers_data_labels), restricted to non-user sectors (DEX, Lending & Yield, Bridges, Oracles, ERC20 Tokens, Stablecoins, NFTs, Infrastructure, RWA). One row per lowercased address. Complements the trace-derived int_execution_accounts_non_user_contracts (which misses label-only contracts like the Balancer Vault); the revenue stream models anti-join BOTH sets so labeled contracts do not count as revenue "users". User sectors and Safe proxies are intentionally excluded so mislabeled user Safes still count as users.
