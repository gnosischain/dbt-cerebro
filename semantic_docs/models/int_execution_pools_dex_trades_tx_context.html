int_execution_pools_dex_trades_tx_context

Thin lookup mapping swap transaction hashes to sender (tx_from) and recipient (tx_to). Only contains transactions that appear in int_execution_pools_dex_trades_raw, keeping the table small (~300–400K distinct hashes per month, ~80 MB). Used by int_execution_pools_dex_trades as a lightweight right-side join instead of hitting the full execution.transactions table inside the same query that also runs ASOF price joins.
