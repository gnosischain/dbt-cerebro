int_execution_address_roles_current

Identity-role pivot: one row per on-chain address with boolean flags for every role it plays across the sectors we track (Safe, GPay, Circles, DeFi pools/lending, validator withdrawal, Dune labels). Powers the Graph Explorer's node-kind badges and auto-discovery of applicable graph profiles when a user seeds from a bare address.

Columns

NameTypeDescription
addressStringLowercase, 0x-prefixed EVM address. Sort key for the ReplacingMergeTree.
circles_avatar_typeStringHuman | Group | Org (from Circles v2 registration).
controls_gpay_walletStringOne of the GP Safes this GA user controls (pay_wallet address).
dune_projectStringProject classification from Dune labels (Safe, Circles, GnosisPay, Balancer, etc.).
has_dune_labelUInt81 if the address appears in int_crawlers_data_labels.
is_circles_avatarUInt81 if the address is a registered Circles v2 avatar.
is_circles_wrapperUInt81 if the address is an ERC20 wrapper token contract for a Circles avatar.
is_ga_userUInt81 if this EOA currently controls at least one GP Safe via its Delay Module (per int_execution_gnosis_app_gpay_wallets heuristic — NOT via safes_current_owners, which returns sentinel 0x…0002 for GP Safes).
is_gpay_walletUInt81 if the address is a Gnosis Pay wallet (Safe with GP setup).
is_lending_userUInt81 if the address has an active Aave V3 / SparkLend lending position.
is_lp_providerUInt81 if the address has provided liquidity to a tracked DEX pool.
is_poolUInt81 if the address is a DEX pool contract tracked in pools_dex_liquidity_events.
is_safeUInt81 if the address is registered as a Safe in contracts_safe_registry.
is_safe_ownerUInt81 if the address appears as an owner of at least one Safe.
is_validator_depositorUInt81 if the address is the trailing-20-bytes extraction of a consensus validator's withdrawal_credentials (only validators with 0x01-type credentials).
pool_protocolStringProtocol label if is_lp_provider=1 (Uniswap V3 / Swapr V3 / Balancer V2 / Balancer V3).
