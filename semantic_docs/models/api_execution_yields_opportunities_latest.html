NameTypeDescription
addressStringContract address. For LPs the pool contract; for lending the reserve token contract. Used for Blockscout links.
as_of_dateDateDate the snapshot is computed as of (max date in the underlying data).
borrow_apyNullable(Float64)Variable borrow APY (lending only, NULL for LPs).
fee_pctNullable(Float64)Pool swap fee as a percentage (e.g. 0.3 = 0.3%). Static for Uniswap V3, latest dynamic fee for Swapr V3 and Balancer V3. NULL for lending markets.

nameStringDisplay name. For LPs the pair and address suffix; for lending the token symbol.
net_apr_7dNullable(Float64)Net APR after LVR (fee_apr_7d + lvr_apr_7d). LP only, NULL for lending.
pool_keyNullable(String)Exact full pool label for LP opportunities, matching the labels used by pool-filtered metrics. NULL for lending markets.
protocolStringProtocol name (e.g. Uniswap V3, Swapr V3, Aave V3).
rate_trend_14dArray(Float64)Last 14 daily yield points ordered oldest to newest. LP rows use fee_apr_7d; lending rows use supply APY.
tokenStringBase token symbol (e.g. WETH, USDC.e).
total_borrowedNullable(Float64)Total borrowed in USD (lending only, NULL for LPs). Derived from total_supplied * utilization_rate / 100.
total_suppliedNullable(Float64)Total supplied in USD (lending only, NULL for LPs). Sum of all deposit positions.
tvlNullable(Float64)Total value locked in USD (LP pools only, NULL for lending).
typeString'LP' or 'Lending'.
utilization_rateNullable(Float64)Lending utilization rate as a percentage (total_borrows / total_supply * 100). Lending only, NULL for LPs.
volume_usd_7dNullable(Float64)Total trading volume in USD over the last 7 days (LP only, NULL for lending).
yield_aprNullable(Float64)Fee APR (LP rows).
yield_apyNullable(Float64)Supply APY (lending rows).
