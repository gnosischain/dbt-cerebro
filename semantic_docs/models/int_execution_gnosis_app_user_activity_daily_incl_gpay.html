int_execution_gnosis_app_user_activity_daily_incl_gpay

The composite Gnosis App activity feed (int_execution_gnosis_app_user_activity_daily) EXTENDED with a `gpay_txn` activity_kind sourced from int_execution_gnosis_app_gpay_txns, plus a first-touch onboard for GP-only owners. Consumed only by the "incl. Gnosis Pay" WAU variant (fct_execution_gnosis_app_users_weekly_incl_gpay), so the current metric — and DAU / MAU / WEAU / retention — stay unchanged for comparison. Reads the base activity table ONCE (all activity_kinds) to avoid a ClickHouse predicate-pushdown bug that leaks non-onboard rows into the 'onboard' branch when the same table is read via two opposite-filter CTEs.
