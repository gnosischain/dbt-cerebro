int_execution_yields_user_lp_positions

Per-user LP position summary across all DEX protocols. One row per (provider, pool_address, tick_lower, tick_upper). Aggregates capital in/out from Mint/Burn events, fees from Collect events, computes net V3 liquidity for activity status, and checks in-range against latest pool tick from Swap events.
