stg_consensus__deposits

This model captures deposit transactions from the consensus layer, providing details necessary for analyzing deposit activity and validation processes.

Module: consensus

Status: docs_only

Columns

NameTypeDescription
amountFloat64The amount deposited, typically measured in ETH or the network's native unit.
deposit_indexUInt64A sequential index identifying the deposit within a particular slot.
proofStringProof data supporting the deposit's validity, such as Merkle proof or similar cryptographic evidence.
pubkeyStringThe public key associated with the deposit, representing the depositor's validator identity.
signatureStringCryptographic signature verifying the authenticity of the deposit data.
slotUInt64The slot number indicating the specific block height at which the deposit was processed.
slot_timestampDateTime64The timestamp corresponding to the slot, indicating when the deposit was included in the chain.
withdrawal_credentialsStringCredentials used to authorize withdrawals, linked to the validator's withdrawal address.
