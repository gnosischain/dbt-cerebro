int_execution_gnosis_app_token_offer_claims

Gnosis App token-offer claims — one row per `OfferClaimed` event emitted by any ERC20TokenOfferCycle where the claim was: 1. The caller (`account`) is a Gnosis App user (int_execution_gnosis_app_users_current). 2. The outer tx was relayed by a Cometh V4 bundler (seeds/gnosis_app_relayers, is_active=1). 3. The outer tx.to is the ERC-4337 EntryPoint v0.7 (0x0000000071727de22e5e9d8baf0edac6f37da032). Mirrors the Dune reference query 6582075. Joined to int_execution_gnosis_app_token_offers for offer/cycle/token metadata, and to int_execution_token_prices_daily to price `received` (offer_token) in USD. `spent` is in CRC base units; no USD pricing yet (CRC price feed TBD).
