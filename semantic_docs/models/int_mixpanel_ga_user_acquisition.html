int_mixpanel_ga_user_acquisition

INTERNAL ONLY. Per-user (user_id_hash) marketing attribution. Collapses each identified production user's sparse UTM hits into one stable first-touch and last-touch campaign/source/medium, derived by ordering on event_time (no initial_utm_* super-properties exist in the raw data). Missing values default to 'unknown'. Carries user_id_hash — never exposed to cerebro-api (inherited exclusion) or MCP (expose_to_mcp: false).
