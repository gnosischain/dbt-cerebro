int_execution_safes_module_events

Long-form log of module-state and guard-state mutations on every Safe in contracts_safe_registry. event_kind ∈ (enabled_module, disabled_module, changed_guard, changed_module_guard). target_address is whichever address the event carried: • enabled_module / disabled_module → decoded_params['module'] • changed_guard → decoded_params['guard'] • changed_module_guard → decoded_params['moduleGuard'] decode_logs owns the time-window filter (start_month/end_month + the this-table incremental high-water mark). The outer model only mirrors the strategy switch.
