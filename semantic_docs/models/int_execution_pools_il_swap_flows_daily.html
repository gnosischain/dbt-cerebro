int_execution_pools_il_swap_flows_daily

Daily pool-level base data for LVR computation. Combines swap flows (signed int256 token amounts), fees, TVL, and per-token prices/decimals for Uniswap V3 and Swapr V3 pools. Materialized to avoid deep CTE chains.
