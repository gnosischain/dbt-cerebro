int_execution_prices_dex_ratios

Daily DEX-derived USD prices for whitelist tokens that have no Gnosis Chainlink feed (GBPe, BRLA, BRZ, COW, SAFE). Median over trades where the token is swapped against an oracle-priced anchor token (implied_usd = anchor_amount * anchor_usd / target_amount), requiring >= $1000 notional and >= 5 trades/day. Sourced from the UNPRICED trade tables (pools_dex_trades_raw + stg_cow__trades) to avoid a cycle with the price hub; the USD anchor is int_execution_prices_oracle_daily.
