int_execution_prices_native_daily

Native daily USD price series for whitelist base tokens, assembled from on-chain data to replace the Dune feed: Chainlink oracle prices (authoritative) + sDAI ERC4626 vault rate + sGNO + DEX-derived stablecoins (GBPe/BRLA/BRZ/COW/SAFE, with 30-day rolling-MAD outlier nulling and BRZ<-BRLA fallback), forward-filled across a daily calendar so every symbol is dense. Drop-in for the hub's source CTE; the hub still layers aTokens (lending_market_mapping) and RWA (backedfi) on top. Full-rebuild table. See docs/native_token_prices_build_plan.md.
