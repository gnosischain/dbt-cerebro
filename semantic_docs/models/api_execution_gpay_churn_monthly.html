Monthly user churn breakdown for Gnosis Pay showing user segments and rates by activity scope.

Columns

NameTypeDescription
churn_rateFloat64The percentage of last month's users who churned.
churned_usersUInt64Users active last month but not this month.
monthDateThe first day of the month.
new_usersUInt64Users active for the first time this month.
retained_usersUInt64Users active in both this and the previous month.
retention_rateFloat64The percentage of last month's users who were retained.
returning_usersUInt64Users returning after a period of inactivity.
scopeStringThe activity scope for churn calculation (e.g., all, payment).
total_activeUInt64Total active users this month.
