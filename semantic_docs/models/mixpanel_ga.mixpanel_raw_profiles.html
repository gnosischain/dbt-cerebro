mixpanel_ga.mixpanel_raw_profiles

Daily full snapshot of Mixpanel People/profile properties for the Gnosis App project, ingested via the Engage API by click-runner (ingestors/mixpanel_profiles_ingestor.py). One row per profile per snapshot; ReplacingMergeTree(synced_at) keyed by (project_id, distinct_id) keeps the latest. Profiles are current-state (no history): `pay` is the user's current Gnosis Pay Safe address and initial_utm_* are Mixpanel's set-once first-touch campaign values.
