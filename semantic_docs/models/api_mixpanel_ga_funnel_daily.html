api_mixpanel_ga_funnel_daily

Not exposed via cerebro-api; accessible to the internal MCP / semantic layer. Pass-through over fct_mixpanel_ga_funnel_daily. Day-scoped funnel conversion metrics across three Gnosis App flows. Unit of measurement is uniqExact(user_id_hash) so all rates are DAU-based. Funnels: Onboarding /welcome → ModalJoin open → Passkey login → identified Swap /swap/* visit → SelectSwapAsset → Swap button click Circles /circles → /circles/mint → Success - Circles mint Production-only (app.gnosis.io). Deploy previews and localhost excluded.
