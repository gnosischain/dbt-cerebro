fct_execution_gpay_actions_by_token_weekly

Weekly aggregation of Gnosis Pay activity metrics broken down by action type and token, with cumulative totals.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/gpay/marts/fct_execution_gpay_actions_by_token_weekly.sql

Tags: production execution gpay

Columns

NameTypeDescription
actionStringThe type of activity (e.g., Payment, Deposit, Withdrawal, Cashback).
activity_countUInt64The number of transactions for the action in this week.
activity_count_cumulativeUInt64The cumulative number of transactions up to this week.
tokenStringThe token symbol involved in the activity.
volumeFloat64The total native token volume for the action in this week.
volume_cumulativeFloat64The cumulative native token volume up to this week.
volume_usdFloat64The total USD-equivalent volume for the action in this week.
volume_usd_cumulativeFloat64The cumulative USD-equivalent volume up to this week.
weekDateThe first day of the ISO week for the activity period.
