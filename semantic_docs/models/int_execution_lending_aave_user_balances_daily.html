int_execution_lending_aave_user_balances_daily

Daily per-user aToken balances across Gnosis lending markets (Aave V3 and SparkLend). Reads scaled deltas from int_execution_lending_aave_diffs_daily, builds a calendar of active (protocol, user, reserve) triples, computes running cumulative scaled balances in Int256, then converts to underlying amounts using the end-of-day liquidityIndex. Scoped to reserves in lending_market_mapping. Incremental seeding: on incremental runs the model carries forward the previous max-date balances from as a synthetic delta on max_date+1, so the cumulative running sum stays correct without rescanning history. Full-refresh recomputes from the entire diffs history. scaled_balance and balance_raw are UInt256/Int256 for exact aToken math (mirrors Aave's on-chain WadRayMath); a --full-refresh is required when migrating from the previous Float64 schema so the column types are recreated.
