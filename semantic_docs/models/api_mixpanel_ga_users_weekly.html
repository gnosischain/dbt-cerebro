api_mixpanel_ga_users_weekly

Not exposed via cerebro-api; accessible to the internal MCP / semantic layer. Aggregate-only pass-through view over fct_mixpanel_ga_users_weekly (weekly unique counts, no pseudonyms) — unlike the per-user api_mixpanel_ga_users_daily, this IS safe for MCP exposure. WAU and visitor counts are weekly distincts — do NOT sum across weeks.
