int_execution_token_prices_daily

The `int_execution_token_prices_daily` view consolidates daily price data for various tokens and stablecoins used in execution and trading activities, facilitating analysis of price trends and stability over time.

Module: execution

Status: candidate

Owner: analytics_team

Path: models/execution/prices/intermediate/int_execution_token_prices_daily.sql

Tags: production execution prices daily

Columns

NameTypeDescription
dateDateThe date for which the price data is recorded, formatted as a date.
priceFloat64The recorded price of the token in USD, represented as a floating-point number.
symbolStringThe token or stablecoin symbol. When available, this uses the preferred presentation format from the `tokens_whitelist` seed (e.g. `USDC.e`, `EURe`, `xDAI`).

Dimensions

  • date (time)
  • symbol (categorical)
