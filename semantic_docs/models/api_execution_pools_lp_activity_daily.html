api_execution_pools_lp_activity_daily

API view for daily LP activity (Mint/Burn event counts) per pool per token, unpivoted into one row per event type for the BarChart seriesField. Reuses pool labels and top-pool filtering from fct_execution_pools_daily.

Module: execution

Status: candidate

NameTypeDescription
dateDateCalendar day (UTC) of the LP activity.
labelStringPool label (token pair, protocol, address suffix) from fct_execution_pools_daily.
tokenStringToken symbol the pools are grouped under.
typeStringLP event type — 'Add' (Mint events) or 'Remove' (Burn events).
valueUInt64Count of LP events of this type across all pools containing this token on this day.
