Knowledge Graph Ontology

Schema version: 1   Node types: 14   Edge types: 12   Join edges: 417

Node types

  • address — A raw on-chain account (EOA or contract) identified by a 20-byte address.
  • bridge — A bridge contract moving value between Gnosis Chain and other networks.
  • circles_avatar — A Circles v2 avatar — a human, group, or organisation identity in the Circles trust graph.
  • day — Daily time-spine grain.
  • gpay_wallet — A Gnosis Pay wallet (relayer-owned Safe backing a Gnosis Pay card).
  • month — Monthly time-spine grain.
  • pool — A DEX liquidity pool contract.
  • project_label — A Dune/crawler project or protocol label attached to an address.
  • safe — A Safe (multisig) smart-contract account.
  • sector_day — Sector-by-day composite grain used by cross-sector daily rollups.
  • token — An ERC-20 / token contract address.
  • user_pseudonym — Cross-sector pseudonymous user identifier (pseudonymize_address hash space).
  • validator — A consensus-layer validator, keyed by validator index / withdrawal credentials.
  • week — Weekly (Monday-anchored) time-spine grain.
