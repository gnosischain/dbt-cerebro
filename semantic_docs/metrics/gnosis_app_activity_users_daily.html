gnosis_app_activity_users_daily

Daily distinct-user count per `activity_kind`. Use to size feature-specific engagement (e.g. "how many users swapped on the app last week" vs "how many topped up"). NB: summing across activity_kinds DOUBLE-COUNTS users who did multiple actions — aggregate at the action level if you need a true daily-active-user figure.
